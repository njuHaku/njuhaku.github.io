广义相对论的数学基础（五）：张量

发表于2022-03-28|学习笔记
|字数总计:4.6k|阅读时长:22分钟|阅读量:

张量

多重线性映射

回想单重线性映射：线性空间 XXX 上的线性映射就是 X∗X^*X∗ 中的向量，即这些线性映射也构成一个线性空间。现在考虑双线性映射以及多重线性映射，这些映射是否也构成一个线性空间呢？

双线性映射 (bilinear mapping)
设 X,YX, YX,Y 和 ZZZ 为域 K\mathbb{K}K 上的三个线性空间。若映射 f:X×Y→Zf : X ×Y \rightarrow Zf:X×Y→Z 满足：

∀x1,x2,x∈X, ∀y1,y2,y∈Y, ∀λ1,λ2∈Kf(λ1x1+λ2x2,y)=λ1f(x1,y)+λ2f(x2,y)f(x,λ1y1+λ2y2)=λ1f(x,y1)+λ2f(x,y2)\begin{aligned} & \forall x_1,x_2,x \in X,\ \forall y_1,y_2,y \in Y,\ \forall \lambda_1,\lambda_2 \in \mathbb{K}\\ &f(\lambda_1x_1 + \lambda_2x_2,y) = \lambda_1f(x_1,y) + \lambda_2f(x_2,y)\\ &f(x,\lambda_1y_1 + \lambda_2y_2) = \lambda_1f(x,y_1) + \lambda_2f(x,y_2)\\ \end{aligned} ​∀x1​,x2​,x∈X, ∀y1​,y2​,y∈Y, ∀λ1​,λ2​∈Kf(λ1​x1​+λ2​x2​,y)=λ1​f(x1​,y)+λ2​f(x2​,y)f(x,λ1​y1​+λ2​y2​)=λ1​f(x,y1​)+λ2​f(x,y2​)​

则称 fff 是一个由 X×YX\times YX×Y 到 ZZZ 的双线性映射。

双线性映射 fff 对于 XXX 或 YYY 中的元素分别是线性的。所有这样的双线性映射形成的集合可以表示为 L(2)(X,Y;Z)\mathscr{L}^{(2)}(X,Y;Z)L(2)(X,Y;Z)。

多重线性映射 (multilinear linear)
类似于双线性映射，可以定义如下的 rrr-重线性映射：

f:X1×X2×⋯×Xr→Zf : X_1\times X_2 \times\cdots \times X_r \rightarrow Z f:X1​×X2​×⋯×Xr​→Z

其中 Xi(i=1,⋯ ,r)X_i(i = 1,\cdots,r)Xi​(i=1,⋯,r) 是 rrr 个线性空间。fff 对于每一个 XiX_iXi​ 中的元素都是线性的。同样，所有的 rrr-重线性映射的集合可以表示为：

L(r)(X1,⋅⋅⋅,Xr;Z)\mathscr{L}^{(r)} (X_1,··· ,X_r;Z) L(r)(X1​,⋅⋅⋅,Xr​;Z)

类似于通常的（单重）线性映射的空间 L(1)(X;Z)\mathscr{L}^{(1)}(X;Z)L(1)(X;Z)，我们可以在 L(r)(X1,⋅⋅⋅,Xr;Z)\mathscr{L}^{(r)} (X_1,··· ,X_r;Z)L(r)(X1​,⋅⋅⋅,Xr​;Z) 中引入线性结构：

(f+g)(x1,⋅⋅⋅,xr)=f(x1,⋅⋅⋅,xr)+g(x1,⋅⋅⋅,xr)(λf)(x1,⋅⋅⋅,xr)=λf(x1,⋅⋅⋅,xr)\begin{aligned} &(f + g)(x_1,··· ,x_r) = f(x_1,··· ,x_r) + g(x_1,··· ,x_r)\\ &(\lambda f)(x_1,··· ,x_r) = \lambda f(x_1,··· ,x_r)\\ \end{aligned} ​(f+g)(x1​,⋅⋅⋅,xr​)=f(x1​,⋅⋅⋅,xr​)+g(x1​,⋅⋅⋅,xr​)(λf)(x1​,⋅⋅⋅,xr​)=λf(x1​,⋅⋅⋅,xr​)​

使得 L(r)(X1,⋅⋅⋅,Xr;Z)\mathscr{L}^{(r)} (X_1,··· ,X_r;Z)L(r)(X1​,⋅⋅⋅,Xr​;Z) 成为一个线性空间。

之后提到的张量在本质上就是一个多重线性映射。

张量积

线性空间的张量积

设 X,Y,WX, Y , WX,Y,W 是数域 K\mathbb{K}K 上的三个线性空间。μ:X×Y→W\mu : X × Y \rightarrow Wμ:X×Y→W 是一个双线性映射。若对任意的 K\mathbb{ K}K 上的线性空间 ZZZ 和任意的双线性映射 fff，都存在唯一的线性映射 g:W→Zg : W \rightarrow Zg:W→Z 使得：

f=g∘μf = g \circ \mu f=g∘μ

那么 WWW 就称为是 XXX 和 YYY 的 张量积 ，其中 μ\muμ 称为张量积映射。它把两个线性空间的卡氏积变成它们的张量积。

定理：张量积的存在性和唯一性
设 X,YX, YX,Y 是数域 K\mathbb{K}K 上的两个线性空间。则它们的张量积存在，而且在同构的意义下是唯一的。

接下来我们来构造 线性函数的“张量积”，并且这个张量积空间在同构意义下是唯一的：
设 x∗∈X∗,y∗∈Y∗x^* \in X ^*, y^*\in Y^*x∗∈X∗,y∗∈Y∗ 分别是 XXX 和 YYY 上的线性函数，我们可以定义它们的 张量积，记为 x∗⊗y∗x^* \otimes y^*x∗⊗y∗，其定义如下：

(x∗⊗y∗)(x′,y′)=⟨x′,x∗⟩⟨y′,y∗⟩=x∗(x′)y∗(y′),∀x′∈X, ∀y′∈Y(x ^* \otimes y^* )(x',y' ) = ⟨x' ,x ^* ⟩⟨y',y^* ⟩=x^* (x') y^*(y'),\quad \forall x' \in X,\ \forall y' \in Y (x∗⊗y∗)(x′,y′)=⟨x′,x∗⟩⟨y′,y∗⟩=x∗(x′)y∗(y′),∀x′∈X, ∀y′∈Y

其中 ⟨x′,x∗⟩\langle x',x^*\rangle⟨x′,x∗⟩ 为 配合，将线性空间 XXX 与其对偶空间 X∗X^*X∗ 放在对等的地位：

⟨x′,x∗⟩=x′(x∗)=x∗(x′)⟨x' ,x ^* ⟩ = x'(x^*) = x^*(x') ⟨x′,x∗⟩=x′(x∗)=x∗(x′)

可以看出 x∗⊗y∗x^ * \otimes y^ *x∗⊗y∗ 是 X×YX \times YX×Y 上的双线性函数（由定义容易证明）。即我们有：

x∗⊗y∗∈L(2)(X,Y;K)x ^* \otimes y^ * \in \mathscr{L}^{(2)} (X,Y ;\mathbb{K}) x∗⊗y∗∈L(2)(X,Y;K)

可以很好类比得到 对偶空间的张量积：X∗⊗Y∗X^ * \otimes Y^*X∗⊗Y∗ 是由所有形如 x∗⊗y∗x ^* \otimes y^ *x∗⊗y∗ 的元素张成（线性组合）的线性空间。

注意并非所有 X∗⊗Y∗X^* \otimes Y^*X∗⊗Y∗ 中的元素可以写为 x∗⊗y∗x^* \otimes y^*x∗⊗y∗ 的形式（如量子力学中的纠缠态）。

这个空间中元素的一般形式为：

Ciαe∗i⊗h∗αC_{i\alpha } e^{*i} \otimes h^ {*\alpha } Ciα​e∗i⊗h∗α

其中 {e∗i,i=1,⋅⋅⋅,n}\{e ^{*i} ,i = 1,··· ,n\}{e∗i,i=1,⋅⋅⋅,n} 和 {h∗α,α=1,⋅⋅⋅,m}\{h^{*\alpha } ,\alpha = 1,··· ,m\}{h∗α,α=1,⋅⋅⋅,m} 分别是 X∗X^*X∗ 和 Y∗Y^*Y∗ 中的基（即 XXX 和 YYY 的基底 {ei}\{e_i \}{ei​} 和 {hα}\{h_\alpha \}{hα​} 的对偶基）。

若系数 CiαC_{i\alpha }Ciα​ 可写成 Ciα=aibαC_{i\alpha } = a_i b_\alphaCiα​=ai​bα​ 的形式，则该元素可写成 x∗⊗y∗x^* \otimes y^*x∗⊗y∗ 的形式，我们称其为 可分解的。

由于 {e∗i⊗h∗α,i=1,⋅⋅⋅,n,α=1,⋅⋅⋅,m}\{e^{*i} \otimes h^{*\alpha }, i = 1,···,n,\alpha = 1,···,m\}{e∗i⊗h∗α,i=1,⋅⋅⋅,n,α=1,⋅⋅⋅,m} 是线性独立的，不难得到它们是完备的，构成 X∗⊗Y∗X^* \otimes Y^*X∗⊗Y∗ 的一组基底。故有：

dim⁡(X∗⊗Y∗)=nm\dim(X^* \otimes Y^* ) = nm dim(X∗⊗Y∗)=nm

定理
设 X,YX,YX,Y 是两个线性空间，则：

X∗⊗Y∗=L(2)(X,Y;K)X^* \otimes Y^* = \mathscr{L}^{(2)}(X,Y;\mathbb{K}) X∗⊗Y∗=L(2)(X,Y;K)

由 X∗⊗Y∗X^* \otimes Y^*X∗⊗Y∗ 中的元素均为双重线性映射可知。

考虑到 XXX 与 X∗X^*X∗ 互为对偶，因此可以定义 XXX 与 YYY 的张量积并有：

X⊗Y=L(2)(X∗,Y∗;K)X \otimes Y = \mathscr{L}^{(2)}(X^*,Y^*;\mathbb{K}) X⊗Y=L(2)(X∗,Y∗;K)

可以得到 X⊗YX\otimes YX⊗Y 与 X∗⊗Y∗X^*\otimes Y^*X∗⊗Y∗ 是互为对偶的。

容易看出张量积 ⊗\otimes⊗ 对其左右两个因子都是线性的。因此它定义了一个双线性映射，即 张量积映射：

μ:X×Y→L(2)(X∗,Y∗;K)(x,y)↦x⊗y\begin{aligned} \mu : &X \times Y \rightarrow \mathscr{L}^{(2)} (X^* ,Y^ * ;\mathbb{K})\\ & (x,y) \mapsto x \otimes y\\ \end{aligned} μ:​X×Y→L(2)(X∗,Y∗;K)(x,y)↦x⊗y​

以上我们首先定义了线性空间的张量积，并且指出其在同构意义下是唯一的。由此我们通过线性函数的张量积来构造张量积空间。以下，我们拓展到多重线性函数的情况。

多重线性函数的张量积

设 xxx 和 yyy 分别是 rrr-重线性函数和 sss-重线性函数，即：

x∈L(r)(X1,⋅⋅⋅,Xr;K)y∈L(s)(Y1,⋅⋅⋅,Ys;K)\begin{aligned} &x\in\mathscr{L}^{(r)}(X_ 1 ,··· ,X_r ;\mathbb{K})\\ &y\in\mathscr{L}^{(s)}(Y_1 ,··· ,Y_s ;\mathbb{K})\\ \end{aligned} ​x∈L(r)(X1​,⋅⋅⋅,Xr​;K)y∈L(s)(Y1​,⋅⋅⋅,Ys​;K)​

它们的张量积定义为：

x⊗y(u1,⋅⋅⋅,ur;v1,⋅⋅⋅,vs)=x(u1,⋅⋅⋅,ur)y(v1,⋅⋅⋅,vs)∀ui∈Xi, i=1,⋅⋅⋅,r∀vj∈Yj, j=1,⋅⋅⋅,s\begin{aligned} &x\otimes y(u_1 ,··· ,u_r ;v_1 ,··· ,v_s ) = x(u_1 ,··· ,u_r ) y(v_1 ,··· ,v_s )\\ &\quad \forall u_i \in X_i ,\ i = 1,···,r \quad \forall v_j \in Y_j ,\ j = 1,···,s\\ \end{aligned} ​x⊗y(u1​,⋅⋅⋅,ur​;v1​,⋅⋅⋅,vs​)=x(u1​,⋅⋅⋅,ur​)y(v1​,⋅⋅⋅,vs​)∀ui​∈Xi​, i=1,⋅⋅⋅,r∀vj​∈Yj​, j=1,⋅⋅⋅,s​

多重线性函数的张量积仍是一个多重线性函数，即 x⊗yx \otimes yx⊗y 是一个 (r+s)(r + s)(r+s)-重线性函数：

x⊗y:X1×⋅⋅⋅×Xr×Y1×⋅⋅⋅Ys→Kx \otimes y : X_1 × ··· × X_r × Y_1 × ···Y_s \rightarrow \mathbb{K} x⊗y:X1​×⋅⋅⋅×Xr​×Y1​×⋅⋅⋅Ys​→K

多重线性函数的张量积满足 结合律，即：

(x⊗y)⊗z=x⊗(y⊗z)(x \otimes y) \otimes z = x \otimes (y \otimes z) (x⊗y)⊗z=x⊗(y⊗z)

其中 xxx，yyy 和 zzz 是三个多重线性函数。我们可以无歧义的将三个多重线性函数的张量积写为：

x⊗y⊗zx\otimes y\otimes z x⊗y⊗z

同理，我们也可以定义任意有限个线性空间的张量积。

抽象指标

张量的抽象指标表示是一种在张量上附加指标的一种表示。

例如：对前面的 (r,s)(r,s)(r,s)-型张量 xxx，我们可以将其表示为：

xb1⋯bsa1⋯arx^{a_1 \cdots a_r}_{\qquad b_1 \cdots b_s} xb1​⋯bs​a1​⋯ar​​

指标反映出张量的类型，但并不要求选取某个基底。它不代表 xxx 在某个基底下的分量。

例如: 张量类型 x    cdab     ex^{ab\ \ \ \ \ e}_{\ \ \ \ cd}x    cdab     e​，它是一个多重线性映射：

x:X∗×X∗×X×X×X∗→Kx : X ^* × X ^* × X × X × X ^* \rightarrow \mathbb{K} x:X∗×X∗×X×X×X∗→K

需要指出的是：指标反映类型即可，但在张量的运算或方程中，我们需要考虑到 指标的平衡。

我们将上下同时出现的求和指标称为 哑标，哑标符号的选取是任意的。对于非哑标，要求指标平衡，这体现在：

  • 等式左右出现的指标相同
  • 相同指标的类型相同（上下位置相同）

利用抽象指标，一些张量运算可以很容易的表示。

  • 张量积的抽象指标
    两个张量的张量积 x⊗yx \otimes yx⊗y 可表示为
