自旋

Stern-Gerlach 实验

Stern-Gerlach 实验是德国物理学家奥托·施特恩和瓦尔特·格拉赫为证实原子角动量量子化于1921年到1922年期间完成的一个著名实验 [3]^{[3]}[3]。在该实验中，银原子射线束通过一个不均匀的磁场区域，射线束在磁场作用下发生偏折，最后落在屏上。如果原子磁矩的方向是可以任意取向的，则屏上形成一片黑斑。而实验发现屏上形成了几条清晰的黑斑，表明银原子的磁矩只能取几个特定的方向，从而验证了原子角动量的投影是量子化的。

Fig：Stern-Gerlach 实验[4]^{[4]}[4]

在该实验中，发现了一些无法解释的现象：对于锂、钠、钾、金、银、铜等原子，实验得到了两条斑纹，反推得到原子的角动量应当是 12\frac{1}{2}21​，按照当时的理论，角量子数只能取整数。

为了解释这一点，我们必须引入 自旋 的概念，自旋对应有 自旋角动量。自旋角动量可以取整数值或半整数值。对于电子来说，自旋角动量 S^z\hat{S}_zS^z​ 为 ±12ℏ\pm\frac{1}{2}\hbar±21​ℏ

注意区分自旋与自转的区别，自旋角动量并不意味着是粒子自转产生的角动量，粒子自旋并无经典的对应物，自旋角动量应当理解为粒子的内禀角动量。后来我们会说明：相对论量子力学的狄拉克方程表明电子自旋蕴藏在波函数的旋量结构中。

自旋的描述

对于有自旋的粒子，用波函数描述其状态时。其不但要确定该粒子处于不同空间位置的几率，而且也要确定其自旋具有各种可能取向的几率。如此，波函数不仅依赖于三个连续变量——坐标 x,y,zx,y,zx,y,z，而且依赖于一个离散的 自旋变量 σ\sigmaσ [5]^{[5]}[5]。

下式给出了粒子处于 dVdVdV 体积内，自旋为 σ\sigmaσ 的概率。

∫∣ψ(x,y,z;σ)∣2dV(1)\int |\psi(x,y,z;\sigma)|^2 dV \tag{1} ∫∣ψ(x,y,z;σ)∣2dV(1)

在推导角动量算子的表达式时，我们从空间旋转群出发，从无限小旋转操作导出了角动量算子的代数结构。我们称自旋也为角动量，这是因为自旋算子也满足同角动量算子一样的代数结构：

[S^α,S^β]=iℏεαβγS^γ(2)[\hat{S}_\alpha,\hat{S}_\beta] = i\hbar \varepsilon_{\alpha\beta\gamma} \hat{S}_{\gamma} \tag{2} [S^α​,S^β​]=iℏεαβγ​S^γ​(2)

类比角动量算子，也有如下对易关系：

[S^2,S^α]=0(3)[\hat{S}^2,\hat{S}_{\alpha}] = 0 \tag{3} [S^2,S^α​]=0(3)

这说明，自旋的三个分量是不能够同时确定的，而 S^2\hat{S}^2S^2 与 S^α\hat{S}_{\alpha}S^α​ 是能够同时确定的，它们有共同本征态。以自旋为 sss 的粒子来说，S^2\hat{S}^2S^2 的本征值为 s(s+1)ℏ2s(s+1)\hbar^2s(s+1)ℏ2，S^z\hat{S}_zS^z​ 的本征值为 −s,−s+1,⋅,s−1,s-s,-s+1,\cdot,s-1,s−s,−s+1,⋅,s−1,s，sss 为整数或半整数。（回忆关于角动量分量本征值的表述，那里本征值只能取整数）。对应的自旋为 sss 的粒子，共有 2s+12s+12s+1 个本征态。

对于电子来说，s=12s = \frac{1}{2}s=21​，对应有自旋向上和自旋向下两种状态，我们记为 ∣↑⟩,∣↓⟩|\uparrow\rangle,|\downarrow\rangle∣↑⟩,∣↓⟩。

其中 j=l±12j = l\pm\frac{1}{2}j=l±21​，得到 mj=±j,±(j−1),±(j−2),±32,±12m_j = \pm j,\pm (j-1),\pm (j-2),\pm \frac{3}{2},\pm\frac{1}{2}mj​=±j,±(j−1),±(j−2),±23​,±21​。

考虑到：

J2=(L+S)2=L2+S2+2L⋅S⇒L⋅S=12(J2−L2−S2)\begin{aligned} J^2 &= (L+S)^2\\ &= L^2 + S^2 + 2L\cdot S\\ &\Rightarrow L\cdot S = \frac{1}{2}(J^2 - L^2 - S^2)\\ \end{aligned} J2​=(L+S)2=L2+S2+2L⋅S⇒L⋅S=21​(J2−L2−S2)​

我们可以得到以下的对易关系：

[L⋅S,L2]=[L⋅S,S2]=[L⋅S,J2]=[L⋅S,Jz]=0[L\cdot S,L^2] = [L\cdot S,S^2] = [L\cdot S,J^2] = [L\cdot S,J_z] = 0 [L⋅S,L2]=[L⋅S,S2]=[L⋅S,J2]=[L⋅S,Jz​]=0

如此，我们发现：若哈密顿量中有 L⋅SL\cdot SL⋅S，在耦合基中是能够将哈密顿量对角化的，在非耦合基中这是做不到的。下面举一例。

自旋-轨道耦合作用与碱金属原子光谱的双线结构

原子中的电子绕原子核作空间运动时将产生磁场，这个磁场必定与电子固有磁矩发生作用，使原有能极劈裂并产生附加能移。这就是自旋—轨道耦合作用[1]^{[1]}[1]。

设电子的轨道运动在电子所处位置产生的磁场为 H\bm{H}H。那么其应当等效为以电子为参考系，原子核绕电子回旋产生的磁场，按照毕奥-萨法尔定律得到：

H=−1cv×Zerr3=1cE×v(19)\bm{H} = -\frac{1}{c}\bm{v}\times \frac{Z_e\bm{r}}{r^3} = \frac{1}{c}\bm{E}\times\bm{v} \tag{19} H=−c1​v×r3Ze​r​=c1​E×v(19)

电子的磁矩为：

μ=−eℏ2mcσ\bm{\mu} = -\frac{e\hbar}{2mc}\bm{\sigma} μ=−2mceℏ​σ

可以得到磁矩在磁场中的能量为：

−μ⋅H=eℏ2mcσ⋅(1cE×v)=1m2c2rdVdrL⋅S\begin{aligned} -\bm{\mu}\cdot\bm{H} &= \frac{e\hbar}{2mc}\bm{\sigma} \cdot (\frac{1}{c}\bm{E}\times\bm{v})\\ &= \frac{1}{m^2c^2 r}\frac{dV}{dr}\bm{L}\cdot\bm{S}\\ \end{aligned} −μ⋅H​=2mceℏ​σ⋅(c1​E×v)=m2c2r1​drdV​L⋅S​

上式相比正确的表达式，缺少了 12\frac{1}{2}21​ 这个因子。这项称为 托马斯进动修正（正确考虑洛伦兹变换）。于是，得到正确的轨道-自旋耦合项为：

12mc2rdVdrL⋅S\frac{1}{2mc^2r}\frac{dV}{dr}\bm{L}\cdot\bm{S} 2mc2r1​drdV​L⋅S

这项修正的量级为 β2\beta^2β2，为相对论性修正。

碱金属原子光谱双线结构（如钠黄光）的物理根源正在于，最外层价电子自旋与轨道角动量之间平行反平行耦合使能级出现双重劈裂。此时哈密顿量为：

H=−ℏ22m∇2+V(r)+ξ(r)L⋅SH = -\frac{\hbar^2}{2m}\nabla^2 + V(r) + \xi(r) \bm{L}\cdot\bm{S} H=−2mℏ2​∇2+V(r)+ξ(r)L⋅S

其中 ξ(r)=ℏ22m2c21rdVdr\xi(r) = \frac{\hbar^2}{2m^2c^2} \frac{1}{r}\frac{dV}{dr}ξ(r)=2m2c2ℏ2​r1​drdV​

由此可得轨道-自旋耦合项造成的能移为：

ΔE=12ξnl[j(j+1)−l(l+1)−34]\Delta E = \frac{1}{2}\xi_{nl}[j(j+1) - l(l+1) - \frac{3}{4} ] ΔE=21​ξnl​[j(j+1)−l(l+1)−43​]

其中：

ξnl=∫0∞[Rnl(r)]2ξ(r)r2dr\xi_{nl} = \int_{0}^{\infty}[R_{nl}(r)]^2 \xi(r)r^2dr ξnl​=∫0∞​[Rnl​(r)]2ξ(r)r2dr

可以得到对于自旋-轨道角动量平行耦合与反平行耦合的态，能量变化为：

{ΔE=l2ξnlj=l+12ΔE=−l+12ξnlj=l−12\left\{ \begin{aligned} &\Delta E = \frac{l}{2}\xi_{nl}\quad &j = l+\frac{1}{2}\\ &\Delta E = -\frac{l+1}{2}\xi_{nl}\quad &j = l-\frac{1}{2}\\ \end{aligned} \right. ⎩⎪⎪⎨⎪⎪⎧​​ΔE=2l​ξnl​ΔE=−2l+1​ξnl​​j=l+21​j=l−21​​

耦合基在非耦合基下的展开

考虑到耦合基 χl+12,m+12\chi_{l+\frac{1}{2},m+\frac{1}{2}}χl+21​,m+21​​ 可以使用非耦合基展开为如下形式：

χl+12,m+12=α1Ylmξ++α2Ylm+1ξ−\chi_{l+\frac{1}{2},m+\frac{1}{2}} = \alpha_1Y_{lm}\xi_+ + \alpha_2Y_{lm+1}\xi_- χl+21​,m+21​​=α1​Ylm​ξ+​+α2​Ylm+1​ξ−​

一方面：χl+12,m+12\chi_{l+\frac{1}{2},m+\frac{1}{2}}χl+21​,m+21​​ 作为 J^2\hat{J}^2J^2 的本征态，有

J^2χl+12,m+12=(l+12)(l+32)ℏ2χl+12,m+12(20)\hat{J}^2\chi_{l+\frac{1}{2},m+\frac{1}{2}} = (l+\frac{1}{2})(l+\frac{3}{2})\hbar^2 \chi_{l+\frac{1}{2},m+\frac{1}{2}} \tag{20} J^2χl+21​,m+21​​=(l+21​)(l+23​)ℏ2χl+21​,m+21​​(20)

另一方面：

J^2=(L^+S^)2=L^2+S^2+2L^⋅S^=L^2+S^2+2L^zS^z+L^+S^−+L^−S^+\begin{aligned} \hat{J}^2 &= (\hat{L}+\hat{S})^2\\ &= \hat{L}^2 + \hat{S}^2 + 2\hat{L}\cdot\hat{S}\\ &= \hat{L}^2 + \hat{S}^2 + 2\hat{L}_z\hat{S}_z + \hat{L}_+\hat{S}_{-} + \hat{L}_-\hat{S}_{+}\\ \end{aligned} J^2​=(L^+S^)2=L^2+S^2+2L^⋅S^=L^2+S^2+2L^z​S^z​+L^+​S^−​+L^−​S^+​​

考虑到：

2L^⋅S^(Ylmξ+)=(2L^zS^z+L^+S^−+L^−S^+)(Ylmξ+)=mℏ2Ylmξ++l(l+1)−m(m+1))ℏ2Ylm+1ξ−2L^⋅S^(Ylm+1ξ−)=(2L^zS^z+L^+S^−+L^−S^+)(Ylm+1ξ−)=−(m+1)ℏ2Ylm+1ξ−+l(l+1)−m(m+1)ℏ2Ylmξ+\begin{aligned} 2\hat{L}\cdot\hat{S}(Y_{lm}\xi_+) &= (2\hat{L}_z\hat{S}_z + \hat{L}_+\hat{S}_{-} + \hat{L}_-\hat{S}_{+})(Y_{lm}\xi_+)\\ &= m\hbar^2 Y_{lm}\xi_+ + \sqrt{l(l+1)-m(m+1)})\hbar^2 Y_{lm+1}\xi_-\\ 2\hat{L}\cdot\hat{S}(Y_{lm+1}\xi_-) &= (2\hat{L}_z\hat{S}_z + \hat{L}_+\hat{S}_{-} + \hat{L}_-\hat{S}_{+})(Y_{lm+1}\xi_-)\\ &= -(m+1)\hbar^2 Y_{lm+1}\xi_- + \sqrt{l(l+1)-m(m+1)}\hbar^2 Y_{lm}\xi_+\\ \end{aligned} 2L^⋅S^(Ylm​ξ+​)2L^⋅S^(Ylm+1​ξ−​)​=(2L^z​S^z​+L^+​S^−​+L^−​S^+​)(Ylm​ξ+​)=mℏ2Ylm​ξ+​+l(l+1)−m(m+1)​)ℏ2Ylm+1​ξ−​=(2L^z​S^z​+L^+​S^−​+L^−​S^+​)(Ylm+1​ξ−​)=−(m+1)ℏ2Ylm+1​ξ−​+l(l+1)−m(m+1)​ℏ2Ylm​ξ+​​

于是得到：

J^2χl+12,m+12=[α1(l(l+1)+34+m)+α2l(l+1)−m(m+1)]ℏ2Ylmξ++[α2(l(l+1)+34−(m+1))+α1l(l+1)−m(m+1)]ℏ2Ylm+1ξ−\begin{aligned} \hat{J}^2\chi_{l+\frac{1}{2},m+\frac{1}{2}} =& [\alpha_1(l(l+1)+\frac{3}{4} + m) + \alpha_2\sqrt{l(l+1)-m(m+1)}]\hbar^2Y_{lm}\xi_+\\ &+[\alpha_2(l(l+1)+\frac{3}{4} - (m+1)) + \alpha_1\sqrt{l(l+1)-m(m+1)}]\hbar^2Y_{lm+1}\xi_-\\ \end{aligned} J^2χl+21​,m+21​​=​[α1​(l(l+1)+43​+m)+α2​l(l+1)−m(m+1)​]ℏ2Ylm​ξ+​+[α2​(l(l+1)+43​−(m+1))+α1​l(l+1)−m(m+1)​]ℏ2Ylm+1​ξ−​​

考虑到 (20)(20)(20) 式子，应当有 Ylmξ+Y_{lm}\xi_+Ylm​ξ+​ 与 Ylm+1ξ−Y_{lm+1}\xi_-Ylm+1​ξ−​ 前的系数分别相等。如此，得到化简之后的式子为：

{α2l(l+1)−m(m+1)=α1(l−m)α1l(l+1)−m(m+1)=α2(l+m+1)\left\{ \begin{aligned} & \alpha_2\sqrt{l(l+1)-m(m+1)} = \alpha_1(l-m)\\ & \alpha_1\sqrt{l(l+1)-m(m+1)} = \alpha_2(l+m+1)\\ \end{aligned} \right. {​α2​l(l+1)−m(m+1)​=α1​(l−m)α1​l(l+1)−m(m+1)​=α2​(l+m+1)​
