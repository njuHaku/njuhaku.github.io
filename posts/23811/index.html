固体物理笔记（四）：晶体结合

发表于2022-02-14|学习笔记
|字数总计:6.4k|阅读时长:26分钟|阅读量:

晶体结合

晶体中原子的聚集是原子间相互作用的结果，这种使原子发生聚集的强烈的相互作用就是 化学键[2]^{[2]}[2]。本篇笔记将会讨论不同类型的键合，如金属键、共价键、离子键、范德瓦耳斯键、氢键等。不同的键合导致了凝聚态物质结构形式的差别（但注意，我们区分不同的键只是为思考的便利，实际上，有很多化学键呈现混合或过渡的特征）。

晶体的内聚力来自于原子核、核外电子之间的静电相互作用。对于两原子系统，我们可以将体系的能量写为：

Etot=P122m1+P222m2+q1q24πε0R+E(R)E_{tot} = \frac{P_1^2}{2m_1} + \frac{P_2^2}{2m_2} + \frac{q_1q_2}{4\pi\varepsilon_0 R} + E(R) Etot​=2m1​P12​​+2m2​P22​​+4πε0​Rq1​q2​​+E(R)

其中前三项表示两原子核的动能、原子核之间的静电势能。最后一项为电子的能量。在两原子相互靠近时，不仅会受到核间的库伦斥力，价电子的重新排布也会影响作用力。价电子的重新分布情况取决于原子本身束缚电子的能力，所以，我们以下首先介绍电负性[1]^{[1]}[1]。

原子的电负性

电负性（electronegativity，简写EN），也译作离子性、负电性及阴电性，综合考虑了电离能和电子亲合能，首先由莱纳斯·鲍林于1932年提出[3]^{[3]}[3]。

以下分别介绍电离能与亲和能。

电离能
电离能（Ionization energy），常简记为 EIE_IEI​。指的是将一个电子自一个孤立的原子、离子或分子移至无限远处所需的能量。更广义的用法，第一电离能定义为气态原子失去一个电子成为一价气态正离子所需的最低能量，记作I1I_1I1​；气态一价正离子失去一个电子成为气态二价正离子所需的能量称为第二电离能，记作I2I_2I2​。依此类推[4]^{[4]}[4]。

Fig:第一电离能的周期性变化[5]^{[5]}[5]

电离能衡量一个原子对价电子束缚能量的强弱。

亲和能
电子亲合能（electron affinity，EeaE_{ea}Eea​）的定义是，将使一个电子脱离一个气态的离子或分子所需耗费（释放）的能量。亲和能衡量一个原子束缚外来电子的能力[6]^{[6]}[6]。

电负性综合了电离能与亲和能，Mulliken 建议采用：

χ=12eV(∣E∣+∣I1∣)\chi = \frac{1}{2eV}(|E| + |I_1|) χ=2eV1​(∣E∣+∣I1​∣)

我们常常做一定的线性变换，取用最后得到的相对值。这个相对大小表示元素原子在分子中对成键电子的吸引能力。元素电负性数值越大，原子在形成化学键时对成键电子的吸引力越强。

Pauling，Allred、Rochow 也提出了标度电负性的方法。比较电负性时，注意使用的标度要相同。以下为Pauling标度。

Fig:电负性的周期性变化，Pauling[7]^{[7]}[7]

电负性的周期性变化规律为：

  1. 在一个周期内电负性由左向右不断增强。
  2. 周期表由上到下电负性逐渐减弱。
  3. 周期表越往下，一个周期内的电负性差别越小。

晶体结合的类型

介绍完电负性的概念之后，谈论晶体结合的分类是容易理解的。

金属结合

金属分为碱金属、过渡金属、贵金属。这类元素的电负性通常较小，对价电子的束缚能力很弱，在金属晶体中，价电子呈现出 退局域化 的特征，价电子为整个金属晶体所共有。价电子的公有化将使得电子的能量减小，这也是吸引力的来源。这种键合方式称为 金属键。

Fig: 金属键示例[8]^{[8]}[8]

以一个简单例子来理解退局域化使得电子能量降低：

假设退局域化前，五个电子分别被限制在五个宽度为 aaa 的无穷势阱中，那么得到基态能量为：

E=5ℏ2π22ma2E = \frac{5\hbar^2\pi^2}{2ma^2} E=2ma25ℏ2π2​

退局域化之后，五个电子被限制在一个宽度为 5a5a5a 的无穷势阱中，考虑到泡利不相容原理，基态能量为：

E′=ℏ2π22m(5a)2(2+2⋅4+9)=19ℏ2π250ma2<EE' = \frac{\hbar^2\pi^2}{2m(5a)^2}(2+2\cdot 4 + 9) = \frac{19\hbar^2\pi^2}{50ma^2} < E E′=2m(5a)2ℏ2π2​(2+2⋅4+9)=50ma219ℏ2π2​<E

通过这样一个简单模型的计算，我们得到了认识——退局域化确实能够使得电子能量降低。

金属结合的特点
无方向性

Fig: 凝胶模型

金属晶体的排斥力来自于费米能：

EF=ℏ22m(3π2n0)2/3E_F = \frac{\hbar^2}{2m}(3\pi^2 n_0)^{2/3} EF​=2mℏ2​(3π2n0​)2/3

n0n_0n0​ 电子数密度。体积越小，电子云密度越大，系统的动能增加，同时正离子间的排斥也增大，表现出排斥作用。

金属结合是一种体积效应，对原子排列并无特殊要求，很多金属晶体采取密堆结构(fcc，hcp)，配位数为12；体心立方结构(bcc)也是一种常见的结构，配位数为8。

金属的很多特性，如高导电性、高导热性、范性（延展性）、金属光泽都与金属结合的电子公有化具有密切关系。

共价结合

电负性较强（IVA∼VIIA\mathrm{IVA}\sim \mathrm{VIIA}IVA∼VIIA）的元素，结合成晶体时，多采用共价结合。

首先介绍共价键。

共价键

当中性原子依靠共价电子结合时，必须从每个原子中取一个电子成键，现在我们考虑最简单的例子——氢分子。海特勒（Heitler），伦敦（London）近似求了氢分子的基态能量。

体系的哈密顿量为：

H^=−ℏ22m(∇12+∇22)−e2ra1−e2ra2−e2rb1−e2rb2+e2r12+e2rab\hat{H} = -\frac{\hbar^2}{2m}(\nabla_1^2+\nabla_2^2) -\frac{e^2}{r_{a1}}-\frac{e^2}{r_{a2}}-\frac{e^2}{r_{b1}}-\frac{e^2}{r_{b2}}+\frac{e^2}{r_{12}}+\frac{e^2}{r_{ab}} H^=−2mℏ2​(∇12​+∇22​)−ra1​e2​−ra2​e2​−rb1​e2​−rb2​e2​+r12​e2​+rab​e2​

Fig: 氢分子[9]^{[9]}[9]

共价键的特点：

  1. 饱和性
    一个原子只能形成一定数目的共价键，这决定于未成对电子的数目。

  2. 方向性
    成键方向取决于形成共价键的两个电子轨道的相互重叠程度。

杂化理论

在金刚石中，每一个 C\mathrm{C}C 原子和周围的4个C\mathrm{C}C 原子形成共价键。基态碳原子的电子组态为：1s22s22p21s^22s^22p^21s22s22p2。只有2个未成对电子，为何会形成四根共价键呢？

Pauling 以量子力学为基础提出了杂化轨道理论。其核心思想是将不同原子轨道的叠加重组，从而成为数目相同，能量相等的新轨道。对于金刚石中的 C\mathrm{C}C 原子来说，经历了如下过程：

  1. 从基态 1s22s22p21s^22s^22p^{2}1s22s22p2 →\rightarrow→ 激发态 1s22s12p31s^22s^12p^{3}1s22s12p3 需要能量 4eV4\mathrm{eV}4eV。

这样就产生了四个未配对电子，其中3个在 2p2p2p 轨道上，1个在 2s2s2s 轨道上，这四个轨道叠加形成一个4个等价的 sp3sp^3sp3 杂化轨道。

  1. 成键放出能量 7.6eV7.6\mathrm{eV}7.6eV。因此，净释放能量 ΔE=3.6eV\Delta E=3.6\mathrm{eV}ΔE=3.6eV。

我们发现这种成键方式确实使得体系能量降低了。

这四个 sp3sp^3sp3 轨道为：

这四个 sp3sp^3sp3 轨道的夹角为 109.5°109.5\degree109.5°。

Fig: sp3sp^3sp3 杂化[10]^{[10]}[10]

C,Si,Ge\mathrm{C},\mathrm{Si},\mathrm{Ge}C,Si,Ge 晶体均采用 sp3sp^3sp3 杂化。其配位数为4，一个中心原子与周围4个原子形成一个四面体。

除了 sp3sp^3sp3 杂化之外，还有 sp,sp2,sp3dsp,sp^2, sp^3dsp,sp2,sp3d 等杂化方式，此处不做介绍。

共价键的极性

上述介绍的金刚石和氢分子，形成共价键的原子相同，自然电负性一样，两原子对成键电子对的吸引能力相同，可以预期共价键的正电中心与负电中心重合，我们称这种共价键为 非极性共价键。对应若形成共价键的两原子的电负性不同，如 HCl\mathrm{HCl}HCl 分子，Cl\mathrm{Cl}Cl 对电子的吸引力更强，所以成键电子对向 Cl\mathrm{Cl}Cl 偏移，此时共价键的正、负电荷中心并不重合，我们称这种为 极性共价键。

Fig：共价键的极性&分子的极性

我们要把分子的极性与共价键的极性区分开来，分子的极性取决与整体的正、负电荷中心是否重合。非极性共价键构成的分子一定是非极性分子；极性共价键可以形成极性分子，如 H2O\mathrm{H_2O}H2​O，也可以形成非极性分子，如 CH4\mathrm{CH_4}CH4​。

原子晶体

内部原子以共价键的形式连的晶体为 原子晶体。典型的物质有：金刚石、硅、二氧化硅、碳化硅等。特点是熔沸点很高，硬度大。

离子结合

当两个电负性相差很大的元素，例如 IA:Li,Na\mathrm{IA}:\mathrm{Li},\mathrm{Na}IA:Li,Na 与 VIIA:F,Cl\mathrm{VIIA}:\mathrm{F},\mathrm{Cl}VIIA:F,Cl 结合时，电负性强的元素会完全夺走成键电子对，形成负离子；失去电子的元素成为正离子。其以离子而不是以原子为结合单元。在离子晶体中，正负离子交替排列。

Fig：NaCl晶体，其中绿色代表 Cl−Cl^{-}Cl−，其中紫色代表 Na+Na^{+}Na+ [11]^{[11]}[11]

随着成键原子相差电负性的增加，成键也发生变化：
共价键 →\rightarrow→ 极性共价键 →\rightarrow→ 离子键
需要指出，这种过渡并没有严格的界限。

范德瓦尔斯结合

上述提到的金属键、共价键、离子键都是很强的键，其平均每个原子的结合能大约为几个电子伏特的量级。除此之外，我们发现具有满壳层电子结构的惰性气体原子在形成晶体时，电子结构基本不变，但原子之间也存在一种很弱的相互作用力。称为 范德瓦尔斯 力。

电子的零点运动会产生一个瞬时的电偶极矩，典型的由此下列两种情况：

  1. 电偶极矩同向。原子相互吸引。
  2. 电偶极矩反向。原子相互排斥。

从统计意义上来说，发生情况1的概率要大于2的概率，因此吸引力占优势，这种吸引力就是范德瓦尔斯力。

现在使用一个简化的模型去理解这件事

ω0=cm\omega_0 =\sqrt{\frac{c}{m}} ω0​=mc​​

引入简正坐标：

xa=12(x1+x2),xa=12(x1−x2)x_a = \frac{1}{\sqrt{2}}(x_1 + x_2),\quad x_a = \frac{1}{\sqrt{2}}(x_1-x_2) xa​=2​1​(x1​+x2​),xa​=2​1​(x1​−x2​)

可得

H=[12mps2+12(c−2e2R3)xs2]+[12mpa2+12(c+2e2R3)xa2]H = [\frac{1}{2m}p_s^2 + \frac{1}{2}(c-\frac{2e^2}{R^3}) x_s^2] + [\frac{1}{2m}p_a^2 + \frac{1}{2}(c+\frac{2e^2}{R^3}) x_a^2] H=[2m1​ps2​+21​(c−R32e2​)xs2​]+[2m1​pa2​+21​(c+R32e2​)xa2​]

范德瓦尔斯键是一种弱键，无方向性，无饱和性。惰性原子晶体的结构类似于金属，多采取密堆积结构，除了 He3,He4\mathrm{He^3},\mathrm{He^4}He3,He4 外，全部采取 fcc 结构。

氢键

H\mathrm{H}H 属于 IA\mathrm{IA}IA 族元素，但与其他IA\mathrm{IA}IA 族元素，如锂、钠等相比，电离能要大很多。因此，氢原子难以形成离子键。

元素 第一电离能
氢 13.6eV13.6 \mathrm{eV}13.6eV
锂 5.39eV5.39 \mathrm{eV}5.39eV
钠 5.14eV5.14 \mathrm{eV}5.14eV

在一定条件下，氢原子可以与一个电负性很强的原子形成极性共价键的同时，再与另一个电负性很强（例如 O,F,N\mathrm{O,F,N}O,F,N）的原子形成氢键。

Fig：冰中水分子间的氢键

一个典型的例子是水。氧的价电子组态为 2s22p42s^22p^42s22p4，水分子中的氧原子是 sp3sp^3sp3 杂化，形成 4 个 sp3sp^3sp3 杂化轨道。其中两个轨道上各有一个未成对电子，另外两个轨道上各有一对电子。其中未成对电子与氢原子形成共价键，这个共价键的极性很强。正电中心向氢原子偏移，近乎裸露的质子对相邻氧原子的孤对电子有吸引力，形成了氢键，键能约为 0.1eV0.1\mathrm{eV}0.1eV。

氢键具有饱和性与方向性。在冰以及很多有机分子材料中起重要的作用。

混合键

有很所分子或晶体的键合不单纯的属于上述五种键合方式，而是综合性的。我们以石墨为例进行介绍。

Fig：石墨的结构[12]^{[12]}[12]

石墨中的 C\mathrm{C}C 原子电子组态为 1s22s22p21s^22s^22p^21s22s22p2，采取 sp2sp^2sp2 杂化，得到 333 个 sp2sp^2sp2 杂化轨道与 111 个 2pz2p_z2pz​ 轨道。三个杂化轨道共面，一个 C\mathrm{C}C 原子与平面内三个相邻的 C\mathrm{C}C 原子分别形成共价键。每层 C\mathrm{C}C 原子多余的 2pz2p_z2pz​ 电子，其轨道垂直于杂化轨道形成的平面，这些电子退局域化，形成金属键。层与层之间靠范德瓦尔斯键，结合成三维石墨晶体。

石墨晶体这种复杂的键合方式可以解释其许多性质：在平面内具有良好导电性；层与层之间的作用弱，具有润滑性。

结合能

严格的从薛定谔方程出发求解，求解薛定谔方程是一件很困难的事情。我们在这里期望用唯象方法去对晶体的结合能做一些讨论。

我们考虑内能函数与宏观可观测量之间的关系。根据热力学第一定律：

dU=TdS−pdVdU = TdS - pdV dU=TdS−pdV

在零温时，有：

dU=−pdVdU = -pdV dU=−pdV

可得

dUdV∣V0=−p≈0\frac{dU}{dV}|_{V_0} = -p\approx 0 dVdU​∣V0​​=−p≈0

考虑到外压强 ppp 与晶体的内压强相比是很小的，所以在上式中近似有 p≈0p\approx 0p≈0。V0V_0V0​ 为平衡体积。
计算晶体的体积弹性模量模量

κ=−dp(dV/V)=Vd2UdV2\kappa = -\frac{dp}{(dV/V)} = V \frac{d^2 U}{dV^2} κ=−(dV/V)dp​=VdV2d2U​

如果我们能够给出一个晶体内能的表达式，那么通过实验测量平衡体积与弹性模量，就可以确定表达式中的待定参数，这是该唯象理论的研究思路。

下面以离子晶体、惰性气体为例进行说明。

离子晶体的结合能

静电吸引势
离子晶体由交错排列的正负离子构成。以 NaCl\mathrm{NaCl}NaCl 晶体为例，设原点有一个正离子，正负离子间距为 rrr，任意一离子距离原点的距离为：

d=n12+n22+n32d = \sqrt{n_1^2+n_2^2+n_3^2} d=n12​+n22​+n32​​

其中 n1,n2,n3∈Nn_1,n_2,n_3\in \mathbb{N}n1​,n2​,n3​∈N，n1+n2+n3n_1+n_2+n_3n1​+n2​+n3​ 为奇数代表负离子，为偶数代表正离子。

一对离子的平均库伦能：

重叠排斥能
离子的晶体的排斥能起因于满壳层离子之间电子云的重叠。排斥力是一种短程力，我们考虑其形式为：

brn\frac{b}{r^n} rnb​

对于 NaCl\mathrm{NaCl}NaCl 晶体来说，每个离子的配位数为6，由此每对离子的平均排斥能为：

6brn\frac{6b}{r^n} rn6b​

离子晶体结合能
考虑上述Madelung能与重叠排斥能，晶体的内能可以写为：

U=N[−αe24πε0r+6brn]=N[−Ar+Brn]U = N[-\frac{\alpha e^2}{4\pi \varepsilon_0 r} + \frac{6b}{r^n}] = N[-\frac{A}{r} + \frac{B}{r^n}] U=N[−4πε0​rαe2​+rn6b​]=N[−rA​+rnB​]

晶体的总体积与粒子数存在以下关系：

V=N⋅2r3V = N\cdot 2r^3 V=N⋅2r3

可以得到晶体的结合能

  1. 结合能主要取决于吸引势。
  2. 体积弹性模量主要来自于排斥力。

惰性气体晶体的结合能

考虑惰性气体的内能。主要为两部分组成：

  • 吸引势为范德瓦尔斯势，与原子间距的六次方成反比。
  • 排斥势为原子间距足够近时电子云的交叠导致的泡利排斥势。

Fig：Lennard-Jones 势[14]^{[14]}[14]
