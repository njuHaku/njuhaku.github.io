定态微扰论

非简并微扰论

本篇笔记将介绍 “old-fashion” 的微扰理论。这里我们来看定态微扰论。这里我们首先讨论非简并微扰论：就是说要处理的能级是非简并的。微扰论的出发点是为了解决这样一个问题：为了求解如下能量本征值问题：

Hψn=Enψn(1)H\psi_n = E_n\psi_n \tag{1} Hψn​=En​ψn​(1)

其中 HHH 可分为两部分：

H=H0+H1(2)H = H_0 + H_1\tag{2} H=H0​+H1​(2)

H0H_0H0​ 对应的本征值问题是容易求解的，即：

H0ψn(0)=En(0)ψn(0)(3)H_0\psi_n^{(0)} = E_n^{(0)}\psi_n^{(0)}\tag{3} H0​ψn(0)​=En(0)​ψn(0)​(3)

而 H1≪H0H_1\ll H_0H1​≪H0​，我们可以将 H1H_1H1​ 看成微扰。那么我们可以将方程 (3)(3)(3) 给出的本征能量、本征波函数作为方程 (1)(1)(1) 的本征能量、本征波函数的零阶近似。在此基础上，有：

En=En(0)+ΔEn,ΔEn=En(1)+En(2)+En(3)+⋯ψn=ψn(0)+Δψn,Δψn=ψn(1)+ψn(2)+ψn(3)+⋯(4)\begin{aligned} &E_n = E_n^{(0)} + \Delta E_n,\quad \Delta E_n = E_n^{(1)} + E_n^{(2)} + E_n^{(3)} + \cdots\\ &\psi_n = \psi_n^{(0)} + \Delta \psi_n,\quad \Delta \psi_n = \psi_n^{(1)} + \psi_n^{(2)} + \psi_n^{(3)} + \cdots\\ \end{aligned}\tag{4} ​En​=En(0)​+ΔEn​,ΔEn​=En(1)​+En(2)​+En(3)​+⋯ψn​=ψn(0)​+Δψn​,Δψn​=ψn(1)​+ψn(2)​+ψn(3)​+⋯​(4)

其中角标代表对修正的阶数。
将 (4)(4)(4) 式代入 (1)(1)(1) 得到：

(H0+H1)(ψn(0)+ψn(1)+⋯ )=(En(0)+En(1)+⋯ )(ψn(0)+ψn(1)+⋯ )(H_0+H_1)(\psi_n^{(0)} + \psi_n^{(1)} + \cdots) = (E_n^{(0)} + E_n^{(1)} + \cdots)(\psi_n^{(0)} + \psi_n^{(1)} + \cdots) (H0​+H1​)(ψn(0)​+ψn(1)​+⋯)=(En(0)​+En(1)​+⋯)(ψn(0)​+ψn(1)​+⋯)

将上式展开，令左右相同阶的项相等，将得到以下一组方程

{H0ψn(0)=En(0)ψn(0)H1ψn(0)+H0ψn(1)=En(1)ψn(0)+En(0)ψn(1)H1ψn(1)+H0ψn(2)=En(2)ψn(0)+En(1)ψn(1)+En(0)ψn(2)⋯H1ψn(i−1)+H0ψn(i)=∑k=0iEn(k)ψn(i−k)⋯(5)\left\{ \begin{aligned} &H_0\psi_n^{(0)} = E_n^{(0)}\psi_n^{(0)}\\ &H_1\psi_n^{(0)} + H_0\psi_n^{(1)} = E_n^{(1)}\psi_n^{(0)} + E_n^{(0)}\psi_n^{(1)}\\ &H_1\psi_n^{(1)} + H_0\psi_n^{(2)} = E_n^{(2)}\psi_n^{(0)} + E_n^{(1)}\psi_n^{(1)} + E_n^{(0)}\psi_n^{(2)}\\ &\cdots\\ &H_1\psi_n^{(i-1)} + H_0\psi_n^{(i)} = \sum_{k=0}^{i} E_n^{(k)}\psi_n^{(i-k)}\\ &\cdots\\ \end{aligned}\tag{5} \right. ⎩⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎧​​H0​ψn(0)​=En(0)​ψn(0)​H1​ψn(0)​+H0​ψn(1)​=En(1)​ψn(0)​+En(0)​ψn(1)​H1​ψn(1)​+H0​ψn(2)​=En(2)​ψn(0)​+En(1)​ψn(1)​+En(0)​ψn(2)​⋯H1​ψn(i−1)​+H0​ψn(i)​=k=0∑i​En(k)​ψn(i−k)​⋯​(5)

完成对方程 (5)(5)(5) 的迭代求解后，我们就得到了 kkk 阶修正后的本征能量与本征波函数。

我们首先计算一阶修正。考虑方程组 (5)(5)(5) 的第二式，将其写为 Dirac 符号的形式：

使用 ⟨ψn(0)∣\langle \psi_n^{(0)}|⟨ψn(0)​∣ 与 (6)(6)(6) 式左右两边作内积，得到：

⟨ψn(0)∣H1∣ψn(0)⟩+⟨ψn(0)∣H0∣ψn(1)⟩=En(1)+⟨ψn(0)∣En(0)∣ψn(1)⟩(7)\langle \psi_{n}^{(0)}|H_1|\psi_n^{(0)}\rangle + \langle \psi_{n}^{(0)}|H_0|\psi_n^{(1)}\rangle = E_n^{(1)} + \langle \psi_{n}^{(0)}|E_n^{(0)}|\psi_n^{(1)}\rangle\tag{7} ⟨ψn(0)​∣H1​∣ψn(0)​⟩+⟨ψn(0)​∣H0​∣ψn(1)​⟩=En(1)​+⟨ψn(0)​∣En(0)​∣ψn(1)​⟩(7)

立即得到本征能量的一阶修正项：

En(1)=⟨ψn(0)∣H1∣ψn(0)⟩=⟨n∣H1∣n⟩(8)E_n^{(1)} = \langle \psi_{n}^{(0)}|H_1|\psi_n^{(0)}\rangle = \langle n|H_1|n\rangle \tag{8} En(1)​=⟨ψn(0)​∣H1​∣ψn(0)​⟩=⟨n∣H1​∣n⟩(8)

对于非简并的情形，微扰的引入只会使波函数同样发生很小的变化。有：

ψn(1)=∑mam(1)ψm(0)=an(1)ψn(0)+∑m≠nam(1)ψm(0)\psi_n^{(1)} = \sum_m a_m^{(1)}\psi_m^{(0)} = a_n^{(1)}\psi_n^{(0)} + \sum_{m\neq n}a_m^{(1)}\psi_m^{(0)} ψn(1)​=m∑​am(1)​ψm(0)​=an(1)​ψn(0)​+m​=n∑​am(1)​ψm(0)​

使用 ⟨ψm(0)∣,m≠n\langle \psi_m^{(0)}|,m\neq n⟨ψm(0)​∣,m​=n 与 (6)(6)(6) 式左右两边作内积，得到：

综上得到，对波函数的一阶修正系数为：

am(1)=⟨m∣H1∣n⟩En(0)−Em(0)(9)a_m^{(1)} = \frac{\langle m|H_1|n\rangle}{E_n^{(0)}-E_m^{(0)}}\tag{9} am(1)​=En(0)​−Em(0)​⟨m∣H1​∣n⟩​(9)

在这里，我们发现了该公式只适用于非简并情形的原因：若该体系是简并的，那么对于简并能级 En(0)=Em(0)E_n^{(0)} = E_m^{(0)}En(0)​=Em(0)​，将会有 am(1)→∞a^{(1)}_m \rightarrow \inftyam(1)​→∞，此时当然不能把微扰对波函数的影响当作一个小量！非简并微扰论会失效，我们之后将讨论简并微扰论。

考虑到：

不妨取：

ψn=ψn(0)+∑m≠nam(1)ψm(0)\psi_n = \psi_n^{(0)} + \sum_{m\neq n}a_m^{(1)}\psi_m^{(0)} ψn​=ψn(0)​+m​=n∑​am(1)​ψm(0)​

H1∣ψn(1)⟩+H0∣ψn(2)⟩=En(2)∣ψn(0)⟩+En(1)∣ψn(1)⟩+En(0)∣ψn(2)⟩(11)H_1|\psi_n^{(1)}\rangle + H_0|\psi_n^{(2)}\rangle = E_n^{(2)}|\psi_n^{(0)}\rangle + E_n^{(1)}|\psi_n^{(1)}\rangle + E_n^{(0)}|\psi_n^{(2)}\rangle \tag{11} H1​∣ψn(1)​⟩+H0​∣ψn(2)​⟩=En(2)​∣ψn(0)​⟩+En(1)​∣ψn(1)​⟩+En(0)​∣ψn(2)​⟩(11)

简并微扰论

接下来我们考虑简并微扰论。

我们现在用下标 mmm 表示简并态，下标 kkk 表示非简并态。那么根据非简并微扰论的结果，将有：

am≫aka_m \gg a_k am​≫ak​

即考虑微扰后的态将成为具有相同能量的简并态的线性组合。即只在简并子空间内进行混合。

那么得到：

ψn=∑nαanα(0)ψnα(0)+∑kak(1)ψk(0)≈∑nαanα(0)ψnα(0)\begin{aligned} \psi_n &= \sum_{n_{\alpha}} a_{n_{\alpha}}^{(0)} \psi_{n_{\alpha}}^{(0)} + \sum_{k} a_{k}^{(1)}\psi_k^{(0)}\\ &\approx \sum_{n_{\alpha}}a_{n_{\alpha}}^{(0)}\psi_{n_{\alpha}}^{(0)} \end{aligned} ψn​​=nα​∑​anα​(0)​ψnα​(0)​+k∑​ak(1)​ψk(0)​≈nα​∑​anα​(0)​ψnα​(0)​​

零阶方程给出不考虑微扰时的本征能量：

H0ψα(0)=En(0)ψα(0)H_0\psi_{\alpha}^{(0)} = E_n^{(0)}\psi_{\alpha}^{(0)} H0​ψα(0)​=En(0)​ψα(0)​

一阶方程可以从下式中得到：

(H0+H1)(∑αaα(0)ψα(0)+ψn(1))=(En(0)+En(1))(∑αaα(0)ψα(0)+ψn(1))(H_0+H_1)(\sum_{\alpha}a_{\alpha}^{(0)}\psi_{\alpha}^{(0)} + \psi_n^{(1)}) = (E_n^{(0)} + E_n^{(1)})(\sum_{\alpha}a_{\alpha}^{(0)}\psi_{\alpha}^{(0)} + \psi_n^{(1)}) (H0​+H1​)(α∑​aα(0)​ψα(0)​+ψn(1)​)=(En(0)​+En(1)​)(α∑​aα(0)​ψα(0)​+ψn(1)​)

那么令左右两边的一阶项相等，有：

H0ψn(1)+H1∑αaα(0)ψα(0)=En(0)ψn(1)+En(1)∑αaα(0)ψα(0)H_0\psi_n^{(1)} + H_1\sum_{\alpha}a_{\alpha}^{(0)}\psi_{\alpha}^{(0)} = E_n^{(0)}\psi_n^{(1)} + E_n^{(1)}\sum_{\alpha}a_{\alpha}^{(0)}\psi_{\alpha}^{(0)} H0​ψn(1)​+H1​α∑​aα(0)​ψα(0)​=En(0)​ψn(1)​+En(1)​α∑​aα(0)​ψα(0)​

用 Dirac 符号写为：

H0∣ψn(1)⟩+H1∑αaα(0)∣ψα(0)⟩=En(0)∣ψn(1)⟩+En(1)∑αaα(0)∣ψα(0)⟩H_0|\psi_n^{(1)}\rangle + H_1\sum_{\alpha}a_{\alpha}^{(0)}|\psi_{\alpha}^{(0)}\rangle = E_n^{(0)}|\psi_n^{(1)}\rangle + E_n^{(1)}\sum_{\alpha}a_{\alpha}^{(0)}|\psi_{\alpha}^{(0)}\rangle H0​∣ψn(1)​⟩+H1​α∑​aα(0)​∣ψα(0)​⟩=En(0)​∣ψn(1)​⟩+En(1)​α∑​aα(0)​∣ψα(0)​⟩

左右两边同时作用 ⟨ψβ(0)∣\langle \psi^{(0)}_\beta|⟨ψβ(0)​∣，取内积得到：

∑α⟨β∣H1∣α⟩aα(0)=En(1)aβ(0)\sum_{\alpha}\langle\beta |H_1|\alpha\rangle a_{\alpha}^{(0)} = E_n^{(1)}a_{\beta}^{(0)} α∑​⟨β∣H1​∣α⟩aα(0)​=En(1)​aβ(0)​

可以写做：

∑α(⟨β∣H1∣α⟩−En(1)δαβ)aα(0)=0(12)\sum_{\alpha}(\langle\beta |H_1|\alpha\rangle - E_n^{(1)}\delta_{\alpha\beta} )a_{\alpha}^{(0)} = 0\tag{12} α∑​(⟨β∣H1​∣α⟩−En(1)​δαβ​)aα(0)​=0(12)

于是求解考虑微扰后的能级成为求解以上本征值问题。

对于二阶简并微扰来说，可以从下述方程求解能量 En(2)E_n^{(2)}En(2)​：

det⁡∣∑Em(0)≠En(0)⟨β∣H1∣n⟩⟨m∣H1∣α⟩En(0)−Em(0)−En(2)δαβ∣=0(13)\det|\sum_{E_m^{(0)}\neq E_n^{(0)}}\frac{\langle\beta|H_1|n\rangle\langle m|H_1|\alpha\rangle}{E_n^{(0)}-E_m^{(0)}} - E_n^{(2)}\delta_{\alpha\beta}| = 0\tag{13} det∣Em(0)​​=En(0)​∑​En(0)​−Em(0)​⟨β∣H1​∣n⟩⟨m∣H1​∣α⟩​−En(2)​δαβ​∣=0(13)

我们来以一个具有两重简并的系统为例。考虑具有能量相同的 ∣0⟩,∣1⟩|0\rangle,|1\rangle∣0⟩,∣1⟩ 两态：

H0∣0⟩=E0∣0⟩H0∣1⟩=E0∣1⟩\begin{aligned} H_0|0\rangle = E_0|0\rangle\\ H_0|1\rangle = E_0|1\rangle\\ \end{aligned} H0​∣0⟩=E0​∣0⟩H0​∣1⟩=E0​∣1⟩​

引入微扰 H1H_1H1​：

H1=(αγeiθγe−iθβ)H_1 = \begin{pmatrix} \alpha & \gamma e^{i\theta}\\ \gamma e^{-i\theta} & \beta\\ \end{pmatrix} H1​=(αγe−iθ​γeiθβ​)

使用一阶简并微扰论计算 ΔE\Delta EΔE，我们需要求解以下方程：

∣α−ΔEγeiθγe−iθβ−ΔE∣=0\begin{vmatrix} \alpha -\Delta E & \gamma e^{i\theta}\\ \gamma e^{-i\theta} & \beta-\Delta E \\ \end{vmatrix} = 0 ∣∣∣∣​α−ΔEγe−iθ​γeiθβ−ΔE​∣∣∣∣​=0

解得：

ΔE±=12[(α+β)±(α−β)2+4γ2]\Delta E_{\pm} = \frac{1}{2}[(\alpha+\beta) \pm \sqrt{(\alpha-\beta)^2 + 4\gamma^2}] ΔE±​=21​[(α+β)±(α−β)2+4γ2​]

因此得到本征能量为：

E±=E0+ΔE±E_{\pm} = E_0 + \Delta E_{\pm} E±​=E0​+ΔE±​

对应的本征矢为：

∣±⟩=γγ2+(ΔE±−α)2[∣0⟩+ΔE±−αγe−iθ∣1⟩]|\pm\rangle = \frac{\gamma}{\sqrt{\gamma^2 + (\Delta E_{\pm} - \alpha)^2}} [|0\rangle + \frac{\Delta E_{\pm} - \alpha}{\gamma}e^{-i\theta}|1\rangle] ∣±⟩=γ2+(ΔE±​−α)2​γ​[∣0⟩+γΔE±​−α​e−iθ∣1⟩]

变分原理

很多时候，相互作用不能看作一个小量，此时微扰论不能很好的发挥作用，解析的求出本征波函数和本征能量不是很现实。现在，在很多有关量子力学体系的数值计算中，我们可以根据变分原理去猜一个试探解，通过不断迭代、优化，能够很好的去计算本征能量了。现在我们来看这件事情。

变分原理 Variational principle 的表述如下：

泛函 E[ψ]=⟨ψ∣H∣ψ⟩⟨ψ∣ψ⟩E[\psi] = \frac{\langle \psi|H|\psi\rangle}{\langle \psi|\psi\rangle}E[ψ]=⟨ψ∣ψ⟩⟨ψ∣H∣ψ⟩​ 只有当 ∣ψ⟩|\psi\rangle∣ψ⟩ 为 HHH 的本征态时取极值。

不难证明：

δE[ψ]=⟨ψ+δψ∣H∣ψ+δψ⟩⟨ψ+δψ∣ψ+δψ⟩−⟨ψ∣H∣ψ⟩⟨ψ∣ψ⟩=⟨ψ∣H∣ψ⟩+⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩⟨ψ∣ψ⟩+⟨δψ∣ψ⟩+⟨ψ∣δψ⟩−⟨ψ∣H∣ψ⟩⟨ψ∣ψ⟩=⟨ψ∣H∣ψ⟩+⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩⟨ψ∣ψ⟩[1−⟨δψ∣ψ⟩+⟨ψ∣δψ⟩⟨ψ∣ψ⟩]−⟨ψ∣H∣ψ⟩⟨ψ∣ψ⟩=⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩⟨ψ∣ψ⟩−⟨ψ∣H∣ψ⟩⟨δψ∣ψ⟩+⟨ψ∣δψ⟩⟨ψ∣ψ⟩=1⟨ψ∣ψ⟩[⟨δψ∣H−Eψ∣ψ⟩+⟨ψ∣H−Eψ∣δψ⟩]\begin{aligned} \delta E[\psi] &= \frac{\langle \psi + \delta\psi|H|\psi + \delta\psi\rangle}{\langle \psi + \delta\psi|\psi + \delta\psi\rangle} - \frac{\langle \psi|H|\psi\rangle}{\langle \psi|\psi\rangle}\\ &= \frac{\langle \psi|H|\psi\rangle + \langle \delta \psi|H|\psi\rangle + \langle \psi|H|\delta\psi\rangle}{\langle \psi|\psi\rangle + \langle \delta\psi|\psi\rangle + \langle \psi|\delta\psi\rangle} - \frac{\langle \psi|H|\psi\rangle}{\langle \psi|\psi\rangle}\\ &= \frac{\langle \psi|H|\psi\rangle + \langle \delta \psi|H|\psi\rangle + \langle \psi|H|\delta\psi\rangle}{\langle \psi|\psi\rangle}[1-\frac{\langle \delta\psi|\psi\rangle + \langle \psi|\delta\psi\rangle}{\langle \psi|\psi\rangle}] - \frac{\langle \psi|H|\psi\rangle}{\langle \psi|\psi\rangle}\\ &= \frac{\langle \delta \psi|H|\psi\rangle + \langle \psi|H|\delta\psi\rangle}{\langle \psi|\psi\rangle} - \langle \psi|H|\psi\rangle\frac{\langle \delta\psi|\psi\rangle + \langle \psi|\delta\psi\rangle}{\langle \psi|\psi\rangle}\\ &= \frac{1}{\langle\psi|\psi\rangle}[\langle \delta\psi|H-E_{\psi}|\psi\rangle + \langle \psi|H-E_{\psi}|\delta\psi\rangle] \end{aligned} δE[ψ]​=⟨ψ+δψ∣ψ+δψ⟩⟨ψ+δψ∣H∣ψ+δψ⟩​−⟨ψ∣ψ⟩⟨ψ∣H∣ψ⟩​=⟨ψ∣ψ⟩+⟨δψ∣ψ⟩+⟨ψ∣δψ⟩⟨ψ∣H∣ψ⟩+⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩​−⟨ψ∣ψ⟩⟨ψ∣H∣ψ⟩​=⟨ψ∣ψ⟩⟨ψ∣H∣ψ⟩+⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩​[1−⟨ψ∣ψ⟩⟨δψ∣ψ⟩+⟨ψ∣δψ⟩​]−⟨ψ∣ψ⟩⟨ψ∣H∣ψ⟩​=⟨ψ∣ψ⟩⟨δψ∣H∣ψ⟩+⟨ψ∣H∣δψ⟩​−⟨ψ∣H∣ψ⟩⟨ψ∣ψ⟩⟨δψ∣ψ⟩+⟨ψ∣δψ⟩​=⟨ψ∣ψ⟩1​[⟨δψ∣H−Eψ​∣ψ⟩+⟨ψ∣H−Eψ​∣δψ⟩]​

其中：

Eψ≡⟨ψ∣H∣ψ⟩E_{\psi} \equiv \langle\psi|H|\psi\rangle Eψ​≡⟨ψ∣H∣ψ⟩

当 ψ\psiψ 为 HHH 的本征态时有：

(H−Eψ)∣ψ⟩=0⟨ψ∣(H−Eψ)=0(14)\begin{aligned} (H-E_{\psi})|\psi\rangle = 0\\ \langle \psi|(H-E_{\psi}) = 0\\ \end{aligned}\tag{14} (H−Eψ​)∣ψ⟩=0⟨ψ∣(H−Eψ​)=0​(14)

此时有：

δE[ψ]=0(15)\delta E[\psi] = 0\tag{15} δE[ψ]=0(15)

反过来，由 δψ\delta \psiδψ 的任意性，也可以得到当 δE[ψ]=0\delta E[\psi] = 0δE[ψ]=0 时，ψ\psiψ 必定为 HHH 的本征态，即：

δE[ψ]=0⟺Hψ=Eψψ(16)\delta E[\psi] = 0 \Longleftrightarrow H\psi = E_{\psi}\psi \tag{16} δE[ψ]=0⟺Hψ=Eψ​ψ(16)

因此，我们通过尝试不同的试探波函数 ψ\psiψ，使得系统的能量尽可能地低，那么该试探波函数就会越接近系统的基态波函数。
