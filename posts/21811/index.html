广义相对论的数学基础（四）：线性空间

线性空间

线性空间 (linear space)
线性空间（矢量空间） XXX 是一个域 K\mathbb{K}K 上的模。这个域 K\mathbb{K}K 可以是实数域 R\mathbb{R}R、复数域 C\mathbb{C}C 等等。

在我们初学线性代数时，我们使用了八个公理去定义线性空间，这其实和 “域 K\mathbb{K}K 上的模” 的内涵是完全一致的。我们来看：

设 u,v∈X,a,b∈Ku,v \in X,a,b\in\mathbb{K}u,v∈X,a,b∈K，eee 为域 K\mathbb{K}K 中的恒元。

  • 域 K\mathbb{K}K 上的模 XXX 是一个 阿贝尔群，存在一个群加法 “+++”，XXX 对于 “+++” 成阿贝尔群：
    • 交换律

    u+v=v+uu + v = v + u u+v=v+u

    • 结合律

    u+v=v+uu + v = v + u u+v=v+u

    • 存在零元

    u+0=uu + 0 = u u+0=u

    • 存在逆元

    u+(−u)=0u + (-u) = 0 u+(−u)=0

  • 外部运算标量乘法 “⋅\cdot⋅” 需要满足：
    • 分配律

    a(u+v)=au+av(a+b)u=au+bu\begin{aligned} &a (u + v) = a u + av\\ &(a + b)u = a u + b u\\ \end{aligned} ​a(u+v)=au+av(a+b)u=au+bu​

    • 标量乘法与域 K\mathbb{K}K 的乘法相容

    (ab)u=a(bu)(ab)u= a(b u) (ab)u=a(bu)

    • 域中恒元 eee 的标量乘法：

    eu=ueu = u eu=u

简而言之，线性空间是一个 K−\mathbb{K}-K−模。

常见的线性空间有 R3\mathbb{R}^3R3：三维向量空间。再比如所有系数为实数的多项式的集合 R[X]\mathbb{R}[X]R[X]，对于通常意义上的多项式加法和标量乘法，R[X]\mathbb{R}[X]R[X] 也构成一个向量空间。

我们通常将线性空间中的元素称为 矢量 (vector) 或 向量。因此线性空间也常称为 矢量空间 (vector space) 或 向量空间。

∀x∈X\forall x\in X∀x∈X，我们可以将其展开为：

x=xiei:=∑i=1nxiei,xi∈Kx = x^i e_i := \sum_{i=1}^{n}x^ie_i,\quad x_i\in\mathbb{K} x=xiei​:=i=1∑n​xiei​,xi​∈K

这里上下重复指标代表缩并，即 Einstein 求和公约。其中 {ei}\{e_i\}{ei​} 为该线性空间的一组基底：

{ei,i=1,2,⋯ ,n=dim⁡(X)}\{e_i,i = 1,2,\cdots,n =\dim(X)\} {ei​,i=1,2,⋯,n=dim(X)}

线性空间的子空间

子空间 (subspace)
设 AAA 是 XXX 的一个子集，若在 XXX 所有的代数操作下 AAA 也是一个线性空间，则 AAA 称为 XXX 的一个线性子空间。

例：
设 V={v1,⋯ ,vm}V = \{v_1,\cdots,v_m\}V={v1​,⋯,vm​} 是 XXX 中任意一组矢量，则这组向量张成的空间 Span(V)\mathrm{Span}(V)Span(V) 是 XXX 的一个线性子空间。
Span(V)\mathrm{Span}(V)Span(V) 定义如下：

Span(V):={zivi∣zi∈K,i=1,⋯ ,m}\mathrm{Span}(V) := \{ z^iv_i| z^i \in \mathbb{K}, i =1,\cdots,m\} Span(V):={zivi​∣zi∈K,i=1,⋯,m}

线性子空间的内直和 (internal direct sum)
设 UUU 和 VVV 是 XXX 的两个子空间，且有 U∩V={0}U\cap V = \{0\}U∩V={0}。则 UUU 和 VVV 的内直和定义为：

U⊕V={u+v∣u∈U,v∈V}U \oplus V = \{u + v|u \in U,v \in V \} U⊕V={u+v∣u∈U,v∈V}

注意只有当 U∩V={0}U \cap V = \{0\}U∩V={0} 时，它们的内直和可定义。

定理
设 UUU 和 VVV 是 XXX 的两个子空间，则 U⊕VU \oplus VU⊕V 也是 XXX 的子空间，且有

dim⁡(U⊕V)=dim⁡(U)+dim⁡(V)\dim(U \oplus V ) = \dim(U) + \dim(V ) dim(U⊕V)=dim(U)+dim(V)

不难证明，注意到 U,VU,VU,V 可以用两组基底 {ei},{ej}\{e_i\},\{e_j\}{ei​},{ej​} 张成，且 U∩V={0}U \cap V = \{0\}U∩V={0} 说明该两组基底是线性无关的。且 {ei}∪{ej}\{e_i\} \cup \{e_j\}{ei​}∪{ej​} 能张成 U⊕VU\oplus VU⊕V。

线性子空间的补空间 (complementary)
设 UUU 和 VVV 是 XXX 的两个子空间，且有

X=U⊕VX = U \oplus V X=U⊕V

则称 VVV 为 UUU 的 补空间。

在上面的定义中，VVV 和 UUU 是相互的。即 U,VU,VU,V 互为补空间。
但需要指出的是：给定一个 UUU，它的补空间并不唯一。例如取 X=R3X = \mathbb{R}^3X=R3，U={(0,y,z)∣y,z∈R}U = \{(0,y,z)|y,z\in \mathbb{R}\}U={(0,y,z)∣y,z∈R}，而其补空间 VVV 不唯一，可以取为：

X={(t,at,bt)∣t∈R}X = \{ (t,at,bt)| t\in \mathbb{R}\} X={(t,at,bt)∣t∈R}

其中 a,ba,ba,b 为给定实数。

线性子空间的外直和 (external direct sum)
设 XXX 和 YYY 是数域 K\mathbb{K}K 上的两个线性空间，在卡氏积 X×YX × YX×Y 上定义如下加法 “+++” 和数乘 “⋅·⋅”：

(x1,y1)+(x2,y2)=cλ⋅(x,y)=(λx,λy)\begin{aligned} &(x_1 ,y_1 ) + (x_2 ,y_2 ) = c\\ &\lambda \cdot (x,y) = (\lambda x,\lambda y)\\ \end{aligned} ​(x1​,y1​)+(x2​,y2​)=cλ⋅(x,y)=(λx,λy)​

其中 x1,x2,x∈X, y1,y2,y∈Y, λ∈Kx_1,x_2 ,x \in X,\ y_1 ,y_2 ,y \in Y,\ \lambda \in \mathbb{K}x1​,x2​,x∈X, y1​,y2​,y∈Y, λ∈K，则 X×YX \times YX×Y 配以 “+++” 和 “⋅\cdot⋅” 成为一个线性空间，称为 XXX 和 YYY 的 (外)直和，并记为 X⊕YX \oplus YX⊕Y，即我们有：

X⊕Y=(X×Y,+,⋅)X\oplus Y = (X \times Y,+,\cdot) X⊕Y=(X×Y,+,⋅)

有时候也将 X⊕Y=(X×Y,+,⋅)X\oplus Y = (X \times Y,+,\cdot)X⊕Y=(X×Y,+,⋅) 中矢量的加法 “+”“+”“+” 也记为 “⊕\oplus⊕”，即有：

(x1,y1)⊕(x2,y2)=(x1+x2,y1+y2)(x_1,y_1)\oplus(x_2,y_2) = (x_1 + x_2 ,y_1 + y_2 ) (x1​,y1​)⊕(x2​,y2​)=(x1​+x2​,y1​+y2​)

定理
设 UUU 和 VVV 是 XXX 的两个线性子空间，且 XXX 等于 UUU 和 VVV 的内直和。现将 UUU 和 VVV 看作两个独立的线性空间，则它们的外直和与 XXX 同构。

其中：数域 K\mathbb{K}K 上的两个线性空间 V,V′V,V'V,V′ 同构是指：存在一个双射 σ:V→V′\sigma: V\rightarrow V'σ:V→V′ 保持加法与数乘的结构：
即 ∀k∈K, u,v∈V\forall k\in \mathbb{K},\ u,v \in V∀k∈K, u,v∈V 有：

σ(u+v)=σ(u)+σ(v)σ(ku)=kσ(u)\begin{aligned} \sigma(u+v) &= \sigma(u)+\sigma(v)\\ \sigma(ku) &= k\sigma(u)\\ \end{aligned} σ(u+v)σ(ku)​=σ(u)+σ(v)=kσ(u)​

线性映射

线性映射 (linear mappings)
设 XXX 和 YYY 是某个域 K\mathbb{K}K 上的两个线性空间。映射 f:X→Yf : X \rightarrow Yf:X→Y 称为线性映射，若：

∀α,β∈K, ∀x,y∈X,f(αx+βy)=αf(x)+βf(y)\forall \alpha,\beta \in \mathbb{K},\ \forall x,y \in X,\quad f(\alpha x + \beta y) = \alpha f(x) + \beta f(y) ∀α,β∈K, ∀x,y∈X,f(αx+βy)=αf(x)+βf(y)

线性映射的核 (kernal)
设 f:X→Yf : X \rightarrow Yf:X→Y 为线性映射，XXX 中的子集 {x∣f(x)=0}\{x|f(x) = 0\}{x∣f(x)=0} 称为 fff 的核。记为 ker⁡f\ker fkerf。

定理
设 f:X→Yf : X \rightarrow Yf:X→Y 为线性映射，则 ker⁡f\ker fkerf 是 XXX 的线性子空间。若 ker⁡f={0}\ker f = \{0\}kerf={0}，则 fff 是单射。

容易证明 ker⁡f\ker fkerf 为 XXX 的线性子空间：
∀ x1,x2∈ker⁡f, ∀ α,β∈K\forall\ x_1,x_2\in \ker f,\ \forall\ \alpha,\beta \in \mathbb{K}∀ x1​,x2​∈kerf, ∀ α,β∈K。由线性映射的性质可得：

f(αx1+βx2)=αf(x1)+βf(x2)=0⇒αx1+βx2∈ker⁡ff(\alpha x_1 + \beta x_2) = \alpha f(x_1) + \beta f(x_2) = 0 \Rightarrow \alpha x_1 + \beta x_2\in \ker f f(αx1​+βx2​)=αf(x1​)+βf(x2​)=0⇒αx1​+βx2​∈kerf

不难证明也将满足线性空间的其他公理。且显然有 ker⁡f⊂X\ker f \subset Xkerf⊂X。

另外：利用反证法，若 fff 不是单射，容易得到存在非零元素属于 ker⁡f\ker fkerf。

线性映射的像 (image)
设 f:X→Yf : X \rightarrow Yf:X→Y 为线性映射，则可将其像记为 imf\mathrm{im} fimf。

定理
设 f:X→Yf : X \rightarrow Yf:X→Y 为线性映射，则 imf\mathrm{im}fimf 是 YYY 的子空间。

定理
设 f:X→Yf : X \rightarrow Yf:X→Y 为线性映射，则：

dim⁡(X)=dim⁡(ker⁡f)+dim⁡(imf)\dim(X) = \dim(\ker f) +\dim(\mathrm{im}f) dim(X)=dim(kerf)+dim(imf)

证明：
设 XXX 的维度为 nnn，ker⁡f\ker fkerf 的维度为 mmm，我们只需要证明 imf\mathrm{im}fimf 的维度为 n−mn-mn−m 即可。由于 dim⁡X=n\dim X = ndimX=n，我们可以找到一个极大线性无关基底组：

{ei,i=1,⋯ ,n}\{e_{i},i = 1,\cdots,n\} {ei​,i=1,⋯,n}

由于 Imf\mathrm{Im} fImf 为线性空间，可以取以下基底组：

{f(ei),i=1,⋯ ,n}\{f(e_{i}),i = 1,\cdots,n\} {f(ei​),i=1,⋯,n}

该基底是完备的，但并不一定是线性独立的。考虑到 ker⁡f\ker fkerf 的维度为 mmm，因此将存在 mmm 个线性独立的方程：

f(amiei)=0=amif(ei)f(a^{i}_{m}e_{i}) = 0 = a^i_m f(e_i) f(ami​ei​)=0=ami​f(ei​)

因此可以得到 {f(ei)}\{f(e_i)\}{f(ei​)} 的维度为 m−nm-nm−n。即有：

dim⁡(X)=dim⁡(ker⁡f)+dim⁡(imf)\dim(X) = \dim(\ker f) +\dim(\mathrm{im}f) dim(X)=dim(kerf)+dim(imf)

对偶空间

线性映射的空间
由 XXX 到 YYY 的所有线性映射形成一个集合。我们将其记为 L(1)(X;Y)\mathscr{L}^{(1)} (X;Y)L(1)(X;Y)，在这个集合上我们可以引入一个内部加法：

(f+g)(x)=f(x)+g(x)(f + g)(x) = f(x) + g(x) (f+g)(x)=f(x)+g(x)

和一个外部标量乘积：

(λf)(x)=λf(x),λ∈K(\lambda f)(x) = \lambda f(x),\quad \lambda \in \mathbb{K} (λf)(x)=λf(x),λ∈K

这样由 XXX 到 YYY 的所有线性映射形成一个集合: L(1)(X;Y)\mathscr{L}^{(1)} (X;Y)L(1)(X;Y)，也形成域 K\mathbb{K}K 上的线性空间。

（代数）对偶空间 (Algerbraic dual space)
取 YYY 为域 K\mathbb{K}K，则 L(1)(X;K)\mathscr{L}^{(1)} (X;\mathbb{K})L(1)(X;K) 是 XXX 上 K\mathbb{K}K 值线性函数形成的空间，称为 XXX 的代数对偶空间。记为 X∗X^*X∗，其中的元素可以记为：x∗,y∗,⋯x^*,y^*,\cdotsx∗,y∗,⋯。

定理
设 XXX 是有限维空间，则 X∗X^*X∗ 和 XXX 的维数相同。

证明：
设 x∈X,f∈X∗x\in X,f\in X^*x∈X,f∈X∗，xxx 可展开为 x=xieix = x^ie_ix=xiei​，则有：

f(x)=xif(ei)f(x) = x^i f(e_i) f(x)=xif(ei​)

因此，一个线性函数在矢量上的作用可完全由其在基底 {ei}\{e_i\}{ei​} 上的作用决定。考虑 X∗X^*X∗ 内的基底为 {e∗i}\{e^{*i}\}{e∗i}，我们完全可以选取一组特殊的基 {e∗i}\{e^{*i}\}{e∗i} 使得：

e∗i(ej)=δji,j=1,⋯ ,ne^{*i}(e_j) = \delta^{i}_{j},\quad j = 1,\cdots,n e∗i(ej​)=δji​,j=1,⋯,n

那么就有：

f(x)=f(ei)xi=f(ei)e∗i(x)f(x) = f(e_i)x^i =f(e_i)e^{*i}(x) f(x)=f(ei​)xi=f(ei​)e∗i(x)

其中：

fi=f(ei)∈Kf_i = f(e_i) \in \mathbb{K} fi​=f(ei​)∈K

因为 xxx 任意，可以得到：

f=fie∗if = f_i e^{*i} f=fi​e∗i

因此可以发现：任意一个给定的线性函数，都可以由 {e∗i,i=1,⋯ ,n}\{e^{*i},i=1,\cdots,n\}{e∗i,i=1,⋯,n} 展开。可以证明这个展开是唯一的。

XXX 与 X∗X^*X∗ 的对偶是相互的。有：

(X∗)∗=X(X^{*})^* = X (X∗)∗=X
