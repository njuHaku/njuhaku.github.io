拓扑流形

流形 (manifold)
设 XXX 是一个 Hausdorff 拓扑空间，∀p∈X\forall p\in X∀p∈X，都有 ppp 的一个邻域 N(p)N(p)N(p) 同胚于 Rn\mathbb{R}^nRn 中的某个开集。则称 XXX 为一个 nnn-维的拓扑流形（或简称流形）。

流形局部像欧式空间

坐标卡 (chart)
流形 XXX 上的坐标卡是一个二元组 (U,φ)(U,\varphi)(U,φ)，其中 UUU 是 XXX 的开集，称为坐标卡的定义域；φ\varphiφ 是一个从 UUU 到 Rn\mathbb{R}^nRn 上某个开集 VVV 的同胚：φ:U→V\varphi: U \rightarrow Vφ:U→V。有时也称其为局部坐标系。

设 p∈Up \in Up∈U，则 φ(p)∈V⊂Rn\varphi(p) \in V \subset \mathbb{R}^nφ(p)∈V⊂Rn 。在 Rn\mathbb{R}^ nRn 中 φ(p)\varphi(p)φ(p) 可以用一组数

(x1(p),⋯ ,xn(p))(x^1 (p),\cdots ,x^ n (p)) (x1(p),⋯,xn(p))

表示。这组数称为点 ppp 在坐标卡 (U,φ)(U,\varphi)(U,φ) 中的坐标 (coordinates)。有时候，人们称 xix^ixi 是坐标函数。它们是 Rn\mathbb{R}^nRn 上的一组映射：

xi:Rn→Ru↦xi(u)=(u)i, ∀u∈Rn\begin{aligned} &x^i : \mathbb{R}^n \rightarrow \mathbb{R}\\ &u \mapsto x^i (u) = (u)^i ,\ \forall u \in \mathbb{R}^n \end{aligned} ​xi:Rn→Ru↦xi(u)=(u)i, ∀u∈Rn​

坐标函数 xix^ixi 的作用是：对 Rn\mathbb{R}^nRn 中的任意一点 uuu，给出它的第 iii 个坐标 (u)i(u)^i(u)i。xi(p)x^i (p)xi(p) 是 xi∘φ(p)x^i\circ \varphi(p)xi∘φ(p) 的简写。比较细致地，我们可以定义复合映射：

微分流形

坐标卡集与微分结构

流形 XXX 上的 CkC^kCk 类 坐标卡集 (altas) 是流形 XXX 上的坐标卡的集合 {(Uα,φα)}\{(U_\alpha ,\varphi_\alpha )\}{(Uα​,φα​)}，并要求这些坐标卡满足下面条件：

  • {Uα}\{U_\alpha \}{Uα​} 是 XXX 的一个覆盖
  • 相容性条件
    由同胚 φα\varphi_ \alphaφα​ 给出的映射

φβ∘φα−1:φα(Uα∩Uβ)→φβ(Uα∩Uβ)\varphi_\beta \circ \varphi^{-1}_\alpha: \varphi _\alpha (U_ \alpha \cap U_\beta ) \rightarrow \varphi_{\beta} (U_{\alpha} \cap U_{\beta} ) φβ​∘φα−1​:φα​(Uα​∩Uβ​)→φβ​(Uα​∩Uβ​)

是 Rn\mathbb{R}^nRn 中开集与开集间的 CkC^kCk 类（同胚）映射，即：

xi↦yi=yi(x1,⋯ ,xn)x^i \mapsto y^i = y^i (x_1 ,\cdots ,x_n ) xi↦yi=yi(x1​,⋯,xn​)

是 nnn 个 CkC^kCk 类函数。这时，我们称这两个坐标卡是 CkC^kCk-相容的。

两个 CkC^kCk 类坐标卡集称为是等价的，若它们的合并还是一个 CkC^kCk 类的坐标卡集。即：{Uα,Uα′}\{U_\alpha ,U_{\alpha'} \}{Uα​,Uα′​} 形成一个新的坐标卡定义域，而 {ϕα,ϕα′}\{\phi_\alpha ,\phi_{\alpha'}\}{ϕα​,ϕα′​} 是相应的同胚。并要求这个新的坐标卡集也是 CkC^kCk 类的。利用这种等价关系可以给出 微分结构 的概念。按上述等价关系，可以得到所有 CkC^kCk 类坐标卡集的等价类。而这个等价类便称为一个 CkC^kCk 类微分结构。

例：
极大坐标卡集
设 XXX 是一个 n−n-n− 维流形，A={(Uα,φα)}\mathscr{A} = \{(U_{\alpha} ,\varphi_{\alpha} )\}A={(Uα​,φα​)} 是它的一个坐标卡集，并满足下面条件：

  • {Uα}\{U_\alpha \}{Uα​} 是 XXX 的一个开覆盖；
  • A\mathscr{A}A 中的任意两个坐标卡是 CkC^kCk-相容的；
  • A\mathscr{A}A 是极大的，即若 (U,φ)(U,\varphi)(U,φ) 是 XXX 的任一个坐标卡且和 A\mathscr{A}A 中每个坐标卡都是 CkC^kCk-相容，那么它必然属于 A\mathscr{A}A。

极大坐标集构成一个微分结构。

微分流形

对于拓扑流形 XXX，赋予其一个 CkC^kCk 类微分结构，便称为一个 CkC^kCk 类 微分流形。若要求对考虑的问题，kkk 足够大 (以后我们遇到的通常是这种情况)，即 C∞C^{\infty}C∞ 类微分流形，又被称为 光滑流形。

一个拓扑流形可以赋予不同的微分结构，得到不同的微分流形。实微分流形是在局部上以 Rn\mathbb{R}^nRn 为模板得到的不同微分流形。

实解析流形 (real analytic)
也记为 CωC^\omegaCω 类微分流形. 要求函数 φβ∘ϕα−1\varphi_{\beta}\circ \phi^{-1}_\alphaφβ​∘ϕα−1​ 是实解析的.

复解析流形 (real analytic)
要求函数 φβ∘φα−1\varphi_{\beta}\circ \varphi^{-1}_\alphaφβ​∘φα−1​ 是全纯的 (holomorphic)或复解析的。

直积流形

我们可以从拓扑流形 XnX^nXn 与 YmY^mYm 上出发构造直积流形。设它们的拓扑分别为 UX\mathscr{U}_{X}UX​ 和 UY\mathscr{U}_YUY​，我们可以赋予直积流形 Zn+m=Xn×YmZ^{n+m} = X^n \times Y^mZn+m=Xn×Ym 乘积拓扑。

对于拓扑流形 XnX^nXn 与 YmY_mYm​，我们可以分别在其上引入微分结构及其相应的坐标卡集 {(UX,φX)}\{(U_X,\varphi_X)\}{(UX​,φX​)} 和 {(UY,φY)}\{(U_Y,\varphi_Y)\}{(UY​,φY​)}，这样 XnX^nXn 和 YmY^mYm 成为两个微分流形。

由坐标卡集 {(UX,φX)}\{(U_{X},\varphi_X)\}{(UX​,φX​)} 和 {(UY,φY)}\{(U_{Y},\varphi_Y)\}{(UY​,φY​)}，我们可以构造 Zn+mZ^{n+m}Zn+m 的坐标卡集 (UZ,φZ)(U_Z,\varphi_Z)(UZ​,φZ​)：该坐标卡的定义域为 UZ=UX×UYU_Z = U_X \times U_YUZ​=UX​×UY​，而相应的映射 φZ\varphi_ZφZ​ 定义为：

φZ(p,q)=(φX(p),φY(q)),∀ (p,q)∈UZ\varphi_Z(p,q) = (\varphi_X(p),\varphi_Y(q)),\quad \forall\ (p,q)\in U_Z φZ​(p,q)=(φX​(p),φY​(q)),∀ (p,q)∈UZ​

这样 Zn+mZ^{n+m}Zn+m 上便有了一个微分结构，这样构造出来的微分流形 Zn+mZ^{n+m}Zn+m 称为 XnX^nXn 和 YmY^mYm 的直积流形。

可微映射

可微函数

可微函数 (differentiable)
微分流形上的函数 fff 在点 ppp 称为是可微的，如果在点 ppp 的某个坐标卡 (U,φ)(U,\varphi)(U,φ)，函数

F=f∘φ−1F = f \circ \varphi^{-1} F=f∘φ−1

在 φ(p)\varphi(p)φ(p) 是可微的。

这个定义不依赖于坐标卡的选取：若 f∘φ−1f\circ \varphi^{-1}f∘φ−1 在 φ(p)\varphi(p)φ(p) 可微，则在 ppp 的任意一个坐标卡 (U~,φ~)(\tilde{U},\tilde{\varphi})(U~,φ~​) 在 φ~(p)\tilde{\varphi}(p)φ~​(p) 也是可微的，这是由于：

f∘φ~−1=(f∘φ−1)∘(φ∘φ~−1)f\circ \tilde{\varphi}^{-1} = (f\circ \varphi^{-1})\circ (\varphi\circ \tilde{\varphi}^{-1}) f∘φ~​−1=(f∘φ−1)∘(φ∘φ~​−1)

也是可微的。

rrr 阶可微函数
函数 fff 称为是 CkC^ kCk 微分流形上点 ppp 处的 CrC^ rCr 函数 (r⩽k)(r \leqslant k)(r⩽k)，如果在 ppp 的某个坐标卡 (U,φ)(U,\varphi)(U,φ)，f∘φ−1f \circ \varphi^ {-1}f∘φ−1 在 φ(p)\varphi(p)φ(p) 是 CrC^ rCr 的。

因 φ∘φ~−1\varphi \circ \tilde{\varphi} ^{-1}φ∘φ~​−1 是 CkC^ kCk 的，所以若想得到坐标卡不依赖的定义，我们要求 r⩽kr \leqslant kr⩽k。

CrC^rCr 类函数
微分流形 XXX 上的函数 f:X→Rf : X \rightarrow \mathbb{R}f:X→R 称为是 CrC^ rCr 的，如果它在 XXX 上的每一点都是 CrC ^rCr 的。微分流形 XXX 上的所有 CrC ^rCr 函数的集合，记为 Fr(X)\mathscr{F}^ r (X)Fr(X)。

可微映射

流形间的映射

设 XnX^nXn 和 YmY^mYm 是两个 CkC^kCk 微分流形，fff 是一个映射：f:Xn→Ymf : X^n \rightarrow Y^mf:Xn→Ym。设 (U,φ)(U,\varphi)(U,φ) 是 XnX_ nXn​ 的一个坐标卡，而 (V,ψ)(V,\psi)(V,ψ) 是 YnY_nYn​ 的一个坐标卡，则映射 F\mathcal{F}F：

F=ψ∘f∘φ−1\mathcal{F} = \psi \circ f \circ \varphi ^{-1} F=ψ∘f∘φ−1

这是一个由 Rn\mathbb{R}^nRn 中的开集到 Rm\mathbb{R}^mRm 中开集的映射。

流形间的映射 fff 称为在点 p∈Xnp\in X^np∈Xn 称为是 CrC^rCr 可微的 (r⩽kr\leqslant kr⩽k)，若映射 F=ψ∘f∘φ−1\mathcal{F} = \psi \circ f \circ \varphi ^{-1}F=ψ∘f∘φ−1 在 φ(p)\varphi(p)φ(p) 是 CrC^rCr 可微的，即：

yI=FI(x1,⋯ ,xn),I=1,⋯ ,my^I = \mathcal{F}^I(x^1,\cdots,x^n),\quad I=1,\cdots,m yI=FI(x1,⋯,xn),I=1,⋯,m

是 mmm 个 CrC^rCr 个可微函数，其中 {xi,i=1,⋯ ,n}\{x^i,i=1,\cdots,n\}{xi,i=1,⋯,n} 是点 ppp 的坐标，而 yI,I=1,⋯ ,my^I,I=1,\cdots,myI,I=1,⋯,m 是点 f(p)f(p)f(p) 的坐标。

其中条件 r⩽kr\leqslant kr⩽k 保证这个可微的定义不依赖于坐标卡的选择。

若映射 f:Xn→Ymf:X^n\rightarrow Y^mf:Xn→Ym 在点 XnX^nXn 的每一点都是 CrC^rCr 可微的，则称 fff 是 XnX^nXn 到 YmY^mYm 的 CrC^rCr 可微映射。

一些例子：

  • 若取 YmY^mYm 为 R\mathbb{R}R，得到了我们前面有关可微函数的定义。
  • 若取 XnX^nXn 为 R\mathbb{R}R 中的一个区间 III，则我们得到了 YmY^mYm 上的 CrC^rCr 类参数曲线。

曲线
设 XXX 是一个 nnn-维微分流形，映射 c:I→X,t↦c(t)c : I \rightarrow X , t \mapsto c(t)c:I→X,t↦c(t) 称为 XXX 的一条曲线。在 XXX 的坐标卡 (U,φ)(U,\varphi)(U,φ) 下，该曲线可表示为：

xi=(φ ∘ c(t))i,i=1,⋯ ,nx ^ i = (\varphi \ \circ\ c(t))^ i ,\quad i = 1,\cdots ,n xi=(φ ∘ c(t))i,i=1,⋯,n

其中 xix^ixi 是这个坐标卡下的坐标。在不引起歧义时，我们也将上面的表达式简写为：

xi=ci(t) or xi=xi(t)x^i = c^i(t) \ or\ x^i = x^i(t) xi=ci(t) or xi=xi(t)

微分同胚 (diffeomorphism)
设 XnX^nXn 和 YnY^nYn是 CkC^kCk 微分流形。若存在一个映射 f:X→Yf : X \rightarrow Yf:X→Y 满足：

  • fff 是双射
  • fff 和 f−1f^{-1}f−1 都是 CrC^ rCr 可微的

则称 XnX^nXn 和 YnY^nYn 互为 CrC^rCr 微分同胚。映射 fff 称为是一个 CrC^ rCr 微分同胚映射，或简称 CrC^ rCr 微分同胚。

若两个微分结构是微分同胚的，则它们具有相同的微分结构（CrC^rCr 微分结构）。但是需要指出，两个同胚的流形不一定是微分同胚的（可能微分结构不一样）。

现在，若无特殊说明，我们假定考虑的微分流形都是光滑的，即 kkk 足够大。所有光滑流形（对象），以及它们之间的光滑映射（态射）形成一个范畴。微分同胚对应于同构态射。

光滑函数的线性空间
光滑流形 XXX 上的所有光滑函数的集合 F∞(X)\mathscr{F}^\infty (X)F∞(X) 形成一个 R\mathbb{R}R 上的线性空间，可以简记为 F(X)\mathscr{F}(X)F(X)。加法和数乘定义为

(f1+f2)(p)=f1(p)+f2(p),(λf)(p)=λf(p)∀f1,f2∈F(X), ∀p∈X, ∀λ∈R\begin{aligned} &(f_ 1 + f_ 2 )(p) = f_ 1 (p) + f_ 2 (p), (λf)(p) = λf(p)\\ &\qquad \forall f_1 ,f_ 2 \in \mathscr{F}(X),\ \forall p \in X ,\ \forall \lambda \in \mathbb{R}\\ \end{aligned} ​(f1​+f2​)(p)=f1​(p)+f2​(p),(λf)(p)=λf(p)∀f1​,f2​∈F(X), ∀p∈X, ∀λ∈R​

光滑函数环
事实上，我们还可以在这个线性空间 F(X)\mathscr{F}(X)F(X) 中定义一个乘法：

(f1f2)(p)=f1(p)f2(p)(f_ 1 f_ 2 )(p) = f_ 1 (p)f_ 2 (p) (f1​f2​)(p)=f1​(p)f2​(p)

使得 F(X)\mathscr{F}(X)F(X) 在这样的加法和乘法下成为一个环。当然它也是实数域 R\mathbb{R}R 上的一个代数。

光滑函数的拖回映射 (pull back)
在微分几何中，拖回是将一个流形上某种结构转移到另一个流形上的一种方法。
设 Φ:X→Y\Phi : X \rightarrow YΦ:X→Y 是一个光滑映射。则映射 Φ\PhiΦ 诱导出 F(Y)\mathscr{F}(Y )F(Y) 到 F(X)\mathscr{F}(X)F(X) 的映射 Φ∗\Phi^*Φ∗ 。若 g∈F(Y)g \in \mathscr{F}(Y )g∈F(Y)，Φ∗g\Phi^* gΦ∗g 定义为 (映射的复合)：

(Φ∗g)(p)=g(Φ(p))=g∘Φ(p)(\Phi^* g)(p) = g(\Phi(p)) = g \circ \Phi(p) (Φ∗g)(p)=g(Φ(p))=g∘Φ(p)

上面关于映射拖回的定义实质上给出了一个由“微分流形和可微映射形成的范畴”到“线性空间和线性映射形成的范畴”的逆变函子。

微分结构和群结构

前面讲到，引入拓扑结构之后，我们可以得到所谓的拓扑群或连续群. 现在除了拓扑结构，我们又有了微分结构。可以预想的，我们会得到一些关于群的可微的概念。在拓扑群上引入与群运算相容的微分结构后，我们便得到所谓的 Lie 群。

李群 (Lie group)
李群 GGG 是一个群，同时又是一个微分流形。其上的微分结构和群结构是相容的，即要求群操作：

G×G→G,(x,y)↦xy−1G \times G \rightarrow G, (x,y) \mapsto xy^{-1} G×G→G,(x,y)↦xy−1

是可微映射 (光滑映射)。

李群同构 (isomorphic)
两个李群称为是同构的，如果他们之间存在一个保持群结构（群同态）的微分同胚。

例如一般线性群 GL(n,R)\mathrm{GL}(n,\mathbb{R})GL(n,R)，容易在上建立坐标卡，使之成为一个微分流形。
